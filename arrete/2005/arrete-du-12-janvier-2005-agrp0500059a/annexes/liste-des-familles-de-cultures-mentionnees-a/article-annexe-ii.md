# Article Annexe II

On entend par familles de cultures au sens de l'article R. 615-12 du code rural les catégories de cultures suivantes :

- les céréales à paille et autres céréales (dont le millet et le sarrasin, l'épeautre, le triticale) ;

- le maïs et le sorgho ;

- les oléagineux ;

- les protéagineux ;

- les légumineuses ;

- les pommes de terre ;

- les betteraves ;

- le tabac ;

- le riz ;

- les plantes à fibre ;

- les légumes de plein champ ;

- les fruits (hors cultures pérennes et pluriannuelles) ;

- les plantes médicinales, florales et aromatiques ;

- les semences ;

- les prairies temporaires.
