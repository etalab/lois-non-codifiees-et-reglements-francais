# Article Annexe IV

A. - Les formes juridiques de l'exploitation du taxi :

- entreprise individuelle ;

- sociétés ;

- statut de l'artisanat ;

- le salariat ;

- la location.

B. - Fiscalité :

- régimes d'imposition et déclarations fiscales :

- sur les bénéfices (artisan, locataire) ;

- sur les revenus (salarié) ;

- taxe à la valeur ajoutée (T.V.A.) :

- définition :

- T.V.A. collectée ;

- T.V.A. récupérable ;

- régularisation ;

- déclarations : rôle de l'expert-comptable ;

- autres taxes liées aux taxis.

C. - Comptabilité :

- connaissances de base permettant d'établir la recette journalière ;

- définitions :

- qu'est-ce qu'un produit d'exploitation ?

- qu'est-ce qu'une charge ?

- qu'est-ce qu'un résultat ?

- obligations comptables :

- tenue de documents :

- livre de recettes ;

- relevé des charges ;

- déclarations annuelles : rôle de l'expert-comptable ;

- pièces comptables :

- factures ;

- quittances d'assurance ;

- carburant (détaxe) ;

- cas du salarié :

- calcul des éléments de rémunération ;

- fiche de paie ;

- déclaration annuelle de revenus.

D. - Les régimes sociaux :

- définitions :

- régime général (locataire, salarié) ;

- régime des artisans ;

- cotisations et prestations par branche :

- qui cotise ? (artisans, locataires, salariés ...) ;

- pour quoi ? (maladie, vieillesse, chômage ...).

E. - Environnement de l'entreprise :

- juridictions compétentes ;

- organisation professionnelle ;

- chambre des métiers.
