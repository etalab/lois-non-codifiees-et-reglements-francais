# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

Pour le fichier des bénéficiaires :

- le nom ;

- le prénom ;

- la date de naissance ;

- le sexe ;

- l'adresse ;

- la date d'attribution du carnet ;

- la date de renouvellement du carnet.

Pour le fichier des associations prestataires de services :

- le nom de l'association ;

- l'adresse ;

- le numéro de téléphone ;

- le nom du correspondant.
