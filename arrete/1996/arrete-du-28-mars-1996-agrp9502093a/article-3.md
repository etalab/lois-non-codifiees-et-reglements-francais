# Article 3

Les taux de la taxe à la première vente prévue à l'article 4 du décret du 28 mars 1996 susvisé sont fixés, par quintal, pour la campagne 1995-1996, ainsi qu'il suit :

15,50 F pour les semences certifiées de céréales hybrides ;

2,47 F pour les semences certifiées de céréales à paille ;

4,04 F pour les semences de base de céréales à paille ;

24,15 F pour les semences de maïs hybrides simples ;

18,90 F pour les semences de maïs hybrides 3 voies ;

14,88 F pour les semences de maïs hybrides doubles ;

27,00 F pour les semences de sorgho ;

9,56 F pour les semences certifiées de vesces, féveroles et sainfoin ;

8,19 F pour les semences certifiées de lupins, pois protéagineux et autres pois fourragers ;

38,72 F pour les semences de graminées fourragères et autres semences fourragères en espèces pures ;

52,52 F pour les semences de graminées fourragères et autres semences fourragères en mélanges ;

62,00 F pour les semences de betteraves sucrières ;

12,94 F pour les semences de betteraves fourragères ;

38,20 F pour les semences de chicorée industrielle ;

7,85 F pour les plants de pommes de terre ;

23,06 F pour les semences de lin ;

53,42 F pour les semences de chanvre ;

24,15 F pour les semences de soja et de ricin ;

49,10 F pour les semences de moutarde commerciale ;

134,00 F pour les semences de tournesol hybride ;

57,93 F pour les semences de tournesol population ;

63,55 F pour les semences de colza hybrides ;

58,33 F pour les autres semences oléagineuses ;

6,83 F pour les semences standard de pois, lentilles et fèves ;

8,66 F pour les semences standard de haricots ;

11,20 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,61 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est égal à 99 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et à 72 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
