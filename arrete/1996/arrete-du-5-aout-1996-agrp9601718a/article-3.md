# Article 3

Les taux de la taxe à la première vente prévue à l'article 4 du décret susvisé sont fixés, par quintal, pour la campagne 1996-1997, ainsi qu'il suit :

15,80 F pour les semences certifiées de céréales hybrides ;

2,52 F pour les semences certifiées de céréales à paille ;

4,12 F pour les semences de base de céréales à paille ;

24,65 F pour les semences de maïs hybrides simples ;

19,30 F pour les semences de maïs hybrides 3 voies ;

15,20 F pour les semences de maïs hybrides doubles ;

27,55 F pour les semences de sorgho ;

9,75 F pour les semences certifiées de vesces, féveroles et sainfoin ;

8,35 F pour les semences certifiées de lupins, pois protéagineux et autres pois fourragers ;

39,50 F pour les semences de graminées fourragères et autres semences fourragères en espèces pures ;

53,30 F pour les semences de graminées fourragères et autres semences fourragères en mélanges ;

63,30 F pour les semences de betteraves sucrières ;

13,20 F pour les semences de betteraves fourragères ;

39,00 F pour les semences de chicorée industrielle ;

8,00 F pour les plants de pommes de terre ;

23,52 F pour les semences de lin ;

54,50 F pour les semences de chanvre ;

24,58 F pour les semences de soja et de ricin ;

49,95 F pour les semences de moutarde commerciale ;

136,50 F pour les semences de tournesol hybride ;

63,55 F pour les semences de colza hybrides ;

59,37 F pour les autres semences oléagineuses ;

6,97 F pour les semences standards de pois, lentilles et fèves ;

8,83 F pour les semences standards de haricots ;

11,42 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,66 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est égal à 101 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et à 74 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
