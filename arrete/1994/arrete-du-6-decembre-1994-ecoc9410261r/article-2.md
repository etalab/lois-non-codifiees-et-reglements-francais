# Article 2

Les catégories d'informations enregistrées sont les suivantes :

- l'unité centrale : code du poste, date de l'observation, site, entité, emplacement, nom utilisateur, modèle, marque et numéro de série du matériel, processeur, écran, mémoires, disques durs, lecteurs, sorties, fichiers autoexec.bat et config.sys ;

- le disque dur : code du poste, date de l'observation, lettre disque, disque local ou réseau, type, nom volume, taille totale disponible et libre, répertoires (fichiers, taille) ;

- les périphériques : code du poste, date de l'observation, code, marque et modèle du périphérique, numéro de série, caractéristiques ;

- les logiciels : code du poste, date de l'observation, lettre du disque, nom, numéro de série, numéro de version du logiciel installé, taille fichiers programme, taille fichiers données.

Les informations enregistrées sont conservées pendant la durée de vie des matériels.
