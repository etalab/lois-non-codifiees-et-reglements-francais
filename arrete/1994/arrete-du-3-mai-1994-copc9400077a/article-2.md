# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

Nom, prénom, date de naissance ;

Adresse, téléphone ;

Activité professionnelle ;

Groupe sanguin ;

Nationalité ;

Examen clinique ;

Résultats d'analyses ;

Diagnostic du médecin ;

Vaccination ;

Médicaments prescrits,
