# Article 2

Chacun des métiers mentionnés à l'article 1er peut se subdiviser dans les spécialités suivantes :

Métiers du bois

Spécialités :

- créateur de prototypes de mobilier ;

- ébéniste ;

- menuisier en sièges.

Métiers du textile

Spécialités :

- artiste en dentelle ;

- artiste licier ;

- artiste teinturier ;

- lingère d'art ;

- rentrayeur en tapis ;

- rentrayeur en tapisserie ;

- créateur et restaurateur de costumes ;

- tapissier décoration ;

- tapissier garniture.

Métiers du papier

Spécialités :

- doreur sur cuir ;

- éditions ;

- monteur en dessin ;

- relieur ;

- restaurateur de papier ou de parchemins ;

- restaurateur d'estampes ;

- restaurateur de globes.

Métiers de l'audiovisuel

Spécialités :

-photographe ;

-multimédia ;

-spécialiste de l'image et du son.

Métiers de la céramique

Spécialité unique.

Métiers des végétaux

Spécialité unique.

Métiers de la présentation des collections

Spécialités :

-aquariologiste ;

-éclairagiste ;

-encadreur-doreur ;

-installateur-monteur d'objets d'art et de documents ;

-peintre-décorateur ;

-métallier.

Métiers des minéraux et métaux

Spécialités :

-argentier d'art ;

-bronzier ;

-créateur de prototypes ;

-fontainier d'art ;

-lustrier ;

-marbrier ;

-restaurateur mouleur de sceaux ;

-restaurateur de monnaies ;

-serrurier d'art.

Métiers des matériaux et volumes

Spécialités :

-maquettiste ;

-moulages ;

-réalisateur de prototypes ;

-volumes.
