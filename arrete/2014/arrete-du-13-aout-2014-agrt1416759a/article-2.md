# Article 2

Le programme d'indemnisation approuvé en application de l'article 1er concerne les départements suivants :

Ardèche (07) ;

Bouches-du-Rhône (13) ;

Haute-Corse (2B) ;

Drôme (26) ;

Gard (30) ;

Hérault (34) ;

Isère (38) ;

Lot-et-Garonne (47) ;

Meurthe-et-Moselle (54) ;

Meuse (55) ;

Moselle (57) ;

Pyrénées-Orientales (66) ;

Bas-Rhin (67) ;

Haut-Rhin (68) ;

Rhône (69) ;

Tarn-et-Garonne (82) ;

Vaucluse (84).
