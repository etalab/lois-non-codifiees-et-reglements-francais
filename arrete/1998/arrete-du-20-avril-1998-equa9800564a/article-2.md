# Article 2

Les formalités de douane, de police et de santé sont assurées en permanence sur les aérodromes suivants :

Ajaccio-Campo dell'Oro ;

Bâle-Mulhouse ;

Bordeaux-Mérignac ;

Fort-de-France - Le Lamentin ;

Lille-Lesquin ;

Lyon-Satolas ;

Marseille-Provence ;

Nice-Côte d'Azur ;

Paris - Charles-de-Gaulle ;

Paris-Le Bourget ;

Paris-Orly ;

Perpignan-Rivesaltes ;

Pointe-à-Pitre - Le Raizet ;

Saint-Denis - Gillot ;

Strasbourg-Entzheim ;

Toulouse-Blagnac.
