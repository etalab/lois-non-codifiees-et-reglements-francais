# Article 2

Les informations recueillies concernent : le sexe, l'année de naissance, le lien avec le locataire, l'installation dans le logement, les caractéristiques physiques du logement, les travaux, les éléments de confort, le mode de location, les aides financières, le paiement des charges, le montant des loyers et des charges.

Les noms et adresses - hormis le code commune - ne sont pas saisis informatiquement.
