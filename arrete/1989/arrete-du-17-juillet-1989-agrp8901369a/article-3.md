# Article 3

Les taux des taxes prévues à l'article 4 du décret n° 87-40 du 26 janvier 1987 susvisé sont fixés, par quintal, ainsi qu'il suit :

12,90 F pour les semences certifiées de céréales hybrides ;

1,75 F pour les semences certifiées de céréales à paille ;

3,10 F pour les semences de base de céréales à paille ;

20,35 F pour les semences de maïs hybrides simples ;

15,80 F pour les semences de maïs hybrides trois voies ;

12,36 F pour les semences de maïs hybrides doubles ;

20,85 F pour les semences de sorgho ;

6,02 F pour les semences certifiées de vesces, féveroles et sainfoin ;

7,45 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

24,35 F pour les semences de graminés fourragères ;

24,35 F pour les autres semences fourragères ;

52,00 F pour les semences de betteraves sucrières ;

9,20 F pour les semences de betteraves fourragères ;

31,00 F pour les semences de chicorée industrielle ;

6,85 F pour les plants de pommes de terre ;

16,75 F pour les semences de lin ;

46,00 F pour les semences de chanvre ;

18,80 F pour les semences de soja ;

37,05 F pour les semences de moutarde commerciale ;

75,20 F pour les semences de tournesol hybride ;

44,70 F pour les autres semences oléagineuses ;

0,54 F pour les semences standard de pois, lentilles et fèves ;

0,54 F pour les semences standard de haricots ;

0,54 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,26 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est au plus égal à 80 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et 57,12 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
