# Article 4

Le taux de la taxe additionnelle prévue à l'article 5 du décret n° 87-40 du 26 janvier 1987 susvisé est fixé, par quintal, ainsi qu'il suit :

0,37 F pour les semences de céréales ;

0,47 F pour les semences de maïs ;

2,34 F pour les semences de sorgho ;

2,23 F pour les semences certifiées de vesces, féveroles et sainfoin ;

1,87 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

9,00 F pour les autres semences fourragères ;

1,92 F pour les semences de betteraves et de chicorée industrielle ;

0,10 F pour les plants de pommes de terre ;

3,06 F pour les semences de lin ;

2,23 F pour les semences de soja ;

43,00 F pour les semences de tournesol ;

5,55 F pour les semences de moutarde commerciale ;

5,55 F pour les autres semences oléagineuses ;

5,38 F pour les semences standard de pois potagers, fèves et lentilles ;

6,95 F pour les semences standard de haricots ;

9,18 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles.

Le taux est égal à 6,12 F par cent unités de semences de betteraves monogermes sucrières et fourragères et de chicorée industrielle présentées en unité de 100 000 graines.
