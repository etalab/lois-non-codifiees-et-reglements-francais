# Article Annexe

Produits :

1. Vins de liqueur, vermouths, apéritifs à base de vin :

Volumes nominaux en litres admis à titre définitif : 0,05 jusqu'à 0,10 ; 0,10 ; 0,20 ; 0,375 ; 0,50 ; 0,75 ; 1 ; 1,5 ; 3 ; 5.

Volumes nominaux en litres admis transitoirement : 0,35 ; 0,70 ; 2.

2. Alcools, eaux-de-vie, liqueurs, préparations alcooliques :

Volumes nominaux en litres admis à titre définitif : 0,02 ; 0,03 ; 0,04 ; 0,05 ; 0,10 (1) ; 0,20 ; 0,35 ; 0,50 ; 0,70 ; 1 ; 1,125 (2) ; 1,5 ; 2 ; 2,5 ; 3 ; 4,5 ; 5 (2) ; 10 (2).

Volumes nominaux en litres admis transitoirement : 0,375 (3) ; 0,75 (3).

(1) Pour les boissons alcooliques additionnées d'eau gazeuse ou de soda, tous volumes inférieurs à 0,10 litre sont également admis.

(2) Valeurs destinées exclusivement à l'usage professionnel.

(3) Jusqu'au 31 décembre 1991.
