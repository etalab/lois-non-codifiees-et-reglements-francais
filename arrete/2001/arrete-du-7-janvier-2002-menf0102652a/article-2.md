# Article 2

8e échelon : 871 ;

7e échelon : 801 ;

6e échelon : 750 ;

5e échelon : 659 ;

4e échelon : 582 ;

3e échelon : 510 ;

2e échelon : 457 ;

1er échelon : 416.
