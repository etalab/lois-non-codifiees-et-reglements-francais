# Article Annexe I

<font size="1">CRITÈRES SPÉCIFIQUES DE RESTRUCTURATION ET DE RECONVERSION DU VIGNOBLE POUR LES PLANTATIONS 2012-2013 EFFECTUÉES DANS LE CADRE DES PLANS COLLECTIFS LOCAUX AGRÉÉS AU TITRE DE LA CAMPAGNE 2010-2011</font>

<font size="1">I. - Critères pour les superficies relevant du conseil de bassin viticole Languedoc-Roussillon auxquelles s'ajoutent les superficies situées dans le département de la Lozère</font>

<font size="1">1. Zones éligibles :</font>

<font size="1">Sont éligibles toutes les superficies situées hors des aires délimitées parcellaires d'appellation d'origine contrôlée auxquelles s'ajoutent les superficies situées sur les aires délimitées parcellaires des appellations d'origine suivantes :<br/>Banyuls, Cabardès, Clairette du Languedoc, Collioure, Corbières, Corbières-Boutenac, Côtes du Roussillon, Côtes du Roussillon Villages, Crémant de Limoux, Faugères, Fitou, Languedoc ou Coteaux du Languedoc, Languedoc Picpoul de Pinet ou Coteaux du Languedoc Picpoul de Pinet, Limoux, Limoux Blanquette de Limoux, Limoux Méthode Ancestrale, Malepère, Minervois, Minervois-La Livinière, Maury, Muscat de Frontignan, Muscat de Lunel, Muscat de Mireval, Muscat de Rivesaltes, Muscat de Saint-Jean-de-Minervois, Rivesaltes, Saint-Chinian.</font>

<font size="1">S'ajoutent les superficies situées dans le département de la Lozère.</font>

<font size="1">Conditions spécifiques :</font>

<font size="1">Pour les superficies situées dans l'aire délimitée parcellaire des AOC " Banyuls " et " Collioure ", le critère de superficie minimale ne s'applique pas à condition que la superficie totale résultant de la restructuration et bénéficiant de l'aide soit au moins égale à 10 ares.</font>

<font size="1">2. Variétés à replanter éligibles :</font>

<font size="1">Les variétés à replanter sur les superficies relevant du conseil de bassin viticole Languedoc-Roussillon et du département de la Lozère sont les suivantes : altesse B, alvarinho B, aranel B, arriloba B, arvine B, bourboulenc B, brun argenté N, cabernet franc N, cabernet-sauvignon N, caladoc N, carmenère N, castets N, chardonnay B, chenin B, clairette B, clairette rose Rs, clarin B, colombard B, cot N, counoise N, couston N, ekigaïna N, fer N, ferradou N, gamaret N, gamay de bouze N, gamay de chaudenay N, gamay N, ganson N, gewurztraminer Rs, gramon N, grenache blanc B, gros manseng B, jurançon noir N, liliorila B, marsanne B, marselan N, mauzac rose Rs, merlot N, mondeuse N, monerac N, morrastel N, mourvèdre N, muscat à petits grains B, muscat à petits grains Rg, muscat d'alexandrie B, muscat de hambourg N, muscat ottonel B, négrette N, nielluccio N, parrellada B, perdea B, petit manseng B, petit verdot N, pinot blanc B, pinot gris G, pinot noir N, piquepoul blanc B, piquepoul gris G, piquepoul noir N, plant de Brunel N, primitivo N, riesling B, rivairenc blanc B, rivairenc gris G, rivairenc N, roussanne B, sauvignon B, sauvignon gris G, segalin N, semebat N, semillon B, sylvaner B, syrah N, tannat N, tourbat B, trousseau N, verdelho B, vermentino B, viognier B.<br/>Les variétés à replanter situées dans le département du Gard et relevant du conseil de bassin viticole vallée du Rhône-Provence sont prévues au point III.</font>

<font size="1">II. ― Critères pour les superficies relevant du conseil de bassin viticole Val de Loire-Centre</font>

<font size="1">Les variétés éligibles à la replantation sont les suivantes :</font>

<font size="1">1. Sur l'ensemble des superficies relevant du conseil de bassin viticole Val de Loire-Centre, pour autant que ces variétés ne soient pas en mesure de permettre une revendication en appellation d'origine sur les parcelles concernées :<br/>Cabernet franc N, cabernet-sauvignon N, chardonnay B, cot N, grolleau N, grolleau gris G, merlot N, pineau d'Aunis N, pinot blanc B, pinot noir N, pinot gris G, sauvignon B, sauvignon gris G.</font>

<font size="1">2. Sur les aires d'appellation d'origine d'Anjou et de Saumur : cabernet franc N, cabernet-sauvignon N, chenin B.</font>

<font size="1">3. Sur l'aire d'appellation d'origine Touraine :<br/>a) Sur l'aire d'appellation d'origine Chinon : chenin B, cabernet franc N ;<br/>b) Sur les aires d'appellation d'origine Bourgueil et Saint-Nicolas-de-Bourgueil : cabernet franc N ;<br/>c) Sur les aires d'appellation d'origine Vouvray et Montlouis-sur-Loire : chenin B ;<br/>d) Sur l'aire d'appellation d'origine Touraine Mesland : cabernet franc N, chardonnay B, chenin B, cot N ;<br/>e) Sur l'aire d'appellation d'origine Touraine Amboise : cot N (sauf les clones 41, 42 et 46), chenin B ;<br/>f) Sur l'aire d'appellation d'origine Touraine Azay-le-Rideau : chenin B, cot N, grolleau N ;<br/>g) Pour les communes de l'aire d'appellation d'origine Touraine Noble Joué : meunier N, pinot gris G, pinot noir N ;<br/>h) Pour les autres communes de l'aire d'appellation d'origine Touraine : cabernet franc N, cot N, sauvignon B, sauvignon gris G.</font>

<font size="1">4. Sur l'aire d'appellation d'origine Cheverny : pinot noir N, sauvignon B.</font>

<font size="1">5. Sur l'aire d'appellation d'origine Cour-Cheverny : romorantin B.</font>

<font size="1">6. Sur l'aire d'appellation d'origine Saint-Pourçain : chardonnay B, pinot noir N, sacy B, sauvignon B.</font>

<font size="1">7. Sur l'aire d'appellation d'origine Côtes d'Auvergne : chardonnay B, gamay N, pinot noir N.</font>

<font size="1">8. Sur l'aire d'appellation d'origine Coteaux du Vendômois : cabernet franc N, chenin B, pineau d'Aunis N, pinot noir N.</font>

<font size="1">9. Sur l'aire d'appellation d'origine Orléans : chardonnay B, meunier N, pinot gris G, pinot noir N.</font>

<font size="1">10. Sur l'aire d'appellation d'origine Orléans-Cléry : cabernet franc N.</font>

<font size="1">11. Sur l'aire d'appellation d'origine Jasnières : chenin B.</font>

<font size="1">12. Sur l'aire d'appellation d'origine Coteaux du Loir : chenin B, pineau d'Aunis N.</font>

<font size="1">13. Sur l'aire d'appellation d'origine Gros Plant du Pays nantais pour des plantations à une densité supérieure à 6 500 pieds par hectare : colombard B, montils B.</font>

<font size="1">14. Sur l'aire d'appellation d'origine Haut-Poitou : cabernet franc N, pinot N, sauvignon B, sauvignon gris G.</font>

<font size="1">15. Sur l'aire d'appellation d'origine Valençay : cot N, pinot noir N, sauvignon B.</font>

<font size="1">III. ― Critères pour les superficies relevant du conseil de bassin viticole vallée du Rhône-Provence</font>

<font size="1">1. Zones éligibles :</font>

<font size="1">1.1. Aire délimitée parcellaire des appellations d'origine :</font>

<font size="1">Sont éligibles toutes les superficies situées sur les aires délimitées parcellaires des appellations d'origine contrôlées suivantes :<br/>Beaumes de Venise, Châtillon-en-Diois, Clairette de Die, Crémant de Die, Clairette de Bellegarde, Costières de Nîmes, Coteaux d'Aix-en-Provence, Grignan-les-Adhémar, Côtes du Rhône (*), Côtes du Rhône Villages (*), Côtes du Vivarais, Languedoc ou Coteaux du Languedoc, Lirac, Luberon, Pierrevert, Rasteau, Saint-Péray, Tavel, Vacqueyras, Ventoux, Vinsobres.<br/>
</font>

<font size="1">(*) Hors des aires délimitées parcellaires plus restreintes. </font>

<font size="1">1.2. Hors aire délimitée parcellaire d'une appellation d'origine :</font>

<font size="1">Sont éligibles toutes les superficies situées hors des aires délimitées parcellaires d'appellation d'origine contrôlée à l'exception des superficies situées dans les départements des Hautes-Alpes, des Alpes-Maritimes et du Var ;</font>

<font size="1">2. Variétés éligibles :</font>

<font size="1">2.1. Sont éligibles sur l'ensemble du bassin viticole les variétés suivantes :</font>

<font size="1">Abouriou N, aléatico N, aligoté B, altesse B, alvarinho B, aranel B, arriloba, arrufiac B, arvine B, auxerrois B, barbaroux Rs, baroque B, biancu gentile B, bourboulenc B, brachet N, cabernet franc N, cabernet-sauvignon N, caladoc N, calitor N, carmenère N, castets N, chardonnay B, chatus N, chenin B, clairette B, clarin B, codivarta B, colombard B, cot N, courbu B, couston N, duras N, ekigaïna N, etraire de la dui N, fer N, ferradou N, gamaret N, gamay de bouze N, gamay de chaudenay N, ganson N, gewurztraminer Rs, gramon N, grenache blanc B, grolleau gris G, grolleau N, gros manseng B, jacquère B, jurançon noir N, len de l'el B, liliorila B, marsanne B, marselan N, mauzac rose Rs, melon B, merlot N, meunier N, mollard N, mondeuse N, monerac N, morrastel N, moussayguès N, mourvèdre N, muscadelle B, muscat à petits grains B, muscat à petits grains Rg, muscat à petits grains Rs, muscat d'alexandrie B, muscat de hambourg N, muscat ottonel B, négrette N, nielluccio N, parrellada B, pascal B, perdea B, persan N, petit courbu B, petit manseng B, petit verdot N, picardan B, pinot blanc B, pinot gris G, pinot noir N, piquepoul gris G, plant de Brunel N, poulsard N, primitivo N, raffiat de moncade B, riesling B, rivairenc blanc B, rivairenc gris G, rivairenc N, rosé du var Rs, roussanne B, sauvignon B, sauvignon gris G, savagnin blanc B, sciaccarello N, segalin N, semebat N, semillon B, sylvaner B, syrah N, tannat N, téoulier N, tibouren N, tourbat B, trousseau N, verdelho B, verdesse B, vermentino B, viognier B.</font>

<font size="1">2.2. Les plantations réalisées sur les aires délimitées parcellaires des appellations d'origine suivantes :<br/>Beaumes de Venise, Coteaux d'Aix-en-Provence, Lirac, Saint-Péray, Tavel, Vacqueyras, Vinsobres sont éligibles uniquement pour les variétés permettant la revendication de l'appellation d'origine concernée.</font>
