# Article 2

Les données à caractère personnel enregistrées relatives aux participants du jeu sont les suivantes :

1° Le nom de famille ;

2° Le prénom ;

3° L'adresse électronique ;

Les données à caractère personnel enregistrées relatives aux gagnants du jeu sont les suivantes :

1° Le nom de famille ;

2° Le prénom ;

3° L'adresse postale personnelle ;

4° Le numéro de téléphone, fixe ou mobile, le cas échéant.
