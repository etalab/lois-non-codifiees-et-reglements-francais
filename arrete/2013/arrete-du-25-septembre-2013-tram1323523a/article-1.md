― latitude : 43° 26 33.81'' N ;

― longitude : 4° 32 32.22'' E.

Point B :

― latitude : 43° 26 12.82'' N ;

― longitude : 4° 33 34.44'' E.

Point C :

― latitude : 43° 24 48.38'' N ;

― longitude : 4° 32 36.04'' E.

Point D :

― latitude : 43° 25 8.56'' N ;
