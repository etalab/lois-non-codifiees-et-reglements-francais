# Article Annexe

1. Les ovoproduits doivent être entreposés dans les locaux appropriés visés au chapitre II, point 1.

2. Les ovoproduits pour lesquels certaines températures d'entreposage sont requises doivent être maintenus à ces températures. Les températures d'entreposage doivent être enregistrées de façon continue, la vitesse de réfrigération doit être telle que le produit atteigne des températures requises aussi rapidement que possible et les récipients doivent être entreposés de telle sorte que l'air puisse circuler librement autour d'eux.

3. Lors de l'entreposage, les températures suivantes ne doivent pas être dépassées :

- pour les produits surgelés : - 18 °C ;

- pour les produits congelés : - 12 °C ;

- pour les produits réfrigérés : +4 °C ;
