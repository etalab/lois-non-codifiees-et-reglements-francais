# Article 2

Les catégories d'informations enregistrées dans ce fichier sont les suivantes :

- nom de l'entreprise ;

- adresse ;

- téléphone ;

- télex ;

- formes juridiques ;

- numéro SIREN ;

- code A.P.E. ;

- responsables ;

- banques ;

- société mère et filiales ;

- emploi et activités ;

- présence d'un syndicat ;

- effectifs et chiffre d'affaires ;

- évolution du capital ;

- prêts bancaires ou fonds publics ;

- comités ;

- procédure judiciaire commerciale.
