# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

- identité des créanciers : nom ou raison sociale, prénom (le cas échéant) ;

- adresse : rue ;

- adresse : ville ;

- code postal ;

- bureau distributeur ;

- numéro de téléphone ;

- références bancaires ou postales ;

- créances (factures, mémoires, etc.) ;

- dépenses payées avant mandatement (essentiellement dépenses de personnel).
