# Article 1

En vue de la constitution des commissions administratives paritaires, les corps et grades des personnels relevant de l'Assistance publique-hôpitaux de Paris sont répartis de la manière suivante :

COMMISSION ADMINISTRATIVE PARITAIRE N° 1

Corps de catégorie A

Groupe 1. - Personnels hospitaliers

Sous-groupe 1. - Psychologues hors classe ; infirmiers généraux de 1re classe ; directeurs d'école de cadres paramédicaux.

Sous-groupe 2. - Psychologues de classe normale ; directeurs d'école préparant au diplôme d'Etat de sage-femme ; directeurs d'école ou de centre préparant aux professions paramédicales ; infirmiers généraux de 2e classe.

Sous-groupe 3. - Sages-femmes surveillantes-chefs.

Sous-groupe 4. - Sages-femmes chefs d'unité ; infirmiers surveillants-chefs ; infirmiers de bloc opératoire surveillants-chefs ; infirmiers anesthésistes surveillants-chefs ; puéricultrices surveillantes-chefs ; techniciens de laboratoire surveillants-chefs ; manipulateurs d'électroradiologie médicale surveillants-chefs ; pédicures podologues surveillants-chefs ; masseurs kinésithérapeutes surveillants-chefs ; ergothérapeutes surveillants-chefs ; psychomotriciens surveillants-chefs ; orthophonistes surveillants-chefs ; orthoptistes surveillants-chefs ; diététiciens surveillants-chefs.

Sous-groupe 5. - Sages-femmes.

Groupe 2. - Personnels administratifs

Chefs de bureau

Groupe 3. - Personnels techniques

Sous-groupe 1. - Ingénieurs en chef.

Sous-groupe 2. - Ingénieurs principaux.

Sous-groupe 3.  - Ingénieurs subdivisionnaires.

COMMISSION ADMINISTRATIVE PARITAIRE N° 2

Corps de catégorie B

Groupe 1. - Personnels infirmiers

Sous-groupe 1. - Infirmiers surveillants ; puéricultrices surveillantes ; infirmiers de bloc opératoire surveillants ; infirmiers anesthésistes surveillants.

Sous-groupe 2. - Infirmiers de classe supérieure ; puéricultrices de classe supérieure ; infirmiers de bloc opératoire de classe supérieure ; infirmiers anesthésistes de classe supérieure.

Sous-Groupe 3. - Infirmiers de classe normale ; puéricultrices de classe normale ; infirmiers de bloc opératoire de classe normale ; infirmiers anesthésistes de classe normale.

Groupe 2. - Personnels médico-techniques

Sous-groupe 1. - Préparateurs en pharmacie de classe fonctionnelle ; techniciens de laboratoire surveillants, manipulateurs d'électroradiologie médicale surveillants ; techniciens de laboratoire de classe fonctionnelle (cadre d'extinction).

Sous-groupe 2. - Techniciens de laboratoire de classe supérieure ; manipulateurs d'électroradiologie médicale de classe supérieure ; techniciens de laboratoire de classe normale, échelons exceptionnels (cadre d'extinction).

Sous-groupe 3 - Techniciens de laboratoire de classe normale ; manipulateurs d'électroradiologie médicale de classe normale ; préparateurs en pharmacie de classe normale ; techniciens de laboratoire de classe normale, 1er au 7e échelon (cadre d'extinction).

Groupe 3. - Personnels de rééducation

Sous-groupe 1. - Pédicures podologues surveillants ; masseurs-kinésithérapeutes surveillants ; ergothérapeutes surveillants ; psychomotriciens surveillants ; orthophonistes surveillants ; orthoptistes surveillants ; diététiciens surveillants.

Sous-groupe 2. - Pédicures podologues de classe supérieure ; masseurs kinésithérapeutes de classe supérieure ; ergothérapeutes de classe supérieure ; psychomotriciens de classe supérieure ; orthophonistes de classe supérieure ; orthoptistes de classe supérieure ; diététiciens de classe supérieure.

Sous-groupe 3. - Pédicures podologues de classe normale ; masseurs kinésithérapeutes de classe normale ; ergothérapeutes de classe normale ; orthophonistes de classe normale ; orthoptistes de classe normale ; diététiciens de classe normale ; psychomotriciens de classe normale.

Groupe 4. - Personnels sociaux

Sous-groupe 1. - Assistantes de service social chefs ; éducateurs spécialisés chefs.

Sous-groupe 2. - Assistantes de service social ; éducateurs spécialisés.

Sous-groupe 3. - Monitrices de jardins d'enfants.

Groupe 5. - Personnels administratifs

Sous-groupe 1. - Adjoints des cadres hospitaliers de classe exceptionnelle ; secrétaires médicaux de classe exceptionnelle.

Sous-groupe 2. - Adjoints des cadres hospitaliers de classe supérieure ; secrétaires médicaux de classe supérieure.

Sous-groupe 3. -  Adjoints des cadres hospitaliers de classe normale ; secrétaires médicaux de classe normale.

Groupe 6. - Personnels techniques et ouvriers

Sous-groupe 1. -  Chefs d'atelier de 1re catégorie.

Sous-groupe 2. - Chefs de laboratoire d'imagerie médicale ; chefs d'atelier de 2e catégorie ; contremaîtres principaux de blanchisserie.

Sous-groupe 3. - Contremaîtres.

COMMISSION ADMINISTRATIVE PARITAIRE N° 3

Corps des catégories C et D

Groupe 1. - Corps des aides-soignants, corps des autres personnels

des services de soins et corps des personnels médico-techniques

Sous-groupe 1. - Aides techniques de laboratoire (cadre d'extinction) ; aides techniques d'électroradiologie (cadre d'extinction) ; aides préparateurs en pharmacie (cadre d'extinction).

Sous-groupe 2. - Aides-soignants de classe supérieure.

Sous-groupe 3. - Aides-soignants de classe normale.

Sous-groupe 4. - Aides d'électroradiologie de classe normale ; aides de laboratoire de classe normale ; aides de pharmacie de classe normale.

Groupe 2. - Corps des agents hospitaliers

Sous-groupe 1. - Agents des services hospitaliers qualifiés ; agents des services hospitaliers hors catégorie.

Sous-groupe 2. - Agents des services hospitaliers.

Groupe 3. - Personnels administratifs

Sous-groupe 1. - Adjoints administratifs hospitaliers principaux ; permanenciers auxiliaires de régulation médicale chefs ; chefs de standard téléphonique principaux.

Sous-groupe 2. - Adjoints administratifs hospitaliers de 1re classe ; chefs de standard téléphonique ; permanenciers auxiliaires de régulation médicale principaux ; secrétaires médicales principales (cadre d'extinction) ; secrétaires médicales et sociales principales (cadre d'extinction) ; inspecteur du service intérieur de 1re classe.

Sous-groupe 3. - Adjoints administratifs hospitaliers de 2e classe ; permanenciers auxiliaires de régulation médicale ; secrétaires médicales (cadre d'extinction) ; secrétaires médicales et sociales (cadre d'extinction).

Sous-groupe 4. - Agents administratifs principaux ; standardistes principaux ; inspecteur du service intérieur de 2e classe.

Sous-groupe 5. - Agents administratifs ; standardistes ; chefs surveillants ; agents de bureau (cadre d'extinction) ; agents de service.

Groupe 4. - Personnels techniques et ouvriers.

Sous-groupe 1. - Magasiniers de 1re catégorie de classe exceptionnelle (cadre d'extinction).

Sous-groupe 2. - Agents techniques principaux ; magasiniers de 1re catégorie de classe normale (cadre d'extinction).

Sous-groupe 3. - Maîtres ouvriers principaux ; conducteurs ambulanciers chefs ; blanchisseurs ; maîtres ouvriers principaux ; premiers prothésistes dentaires.

Sous-groupe 4. - Agents techniques ; maîtres ouvriers ; conducteurs ambulanciers principaux ; blanchisseurs maîtres ouvriers ; prothésistes dentaires ; photographes de 1re catégorie.

Sous-groupe 5. - Conducteurs ambulanciers ; ouvriers de 1re catégorie ; ouvriers professionnels principaux ; blanchisseurs ouvriers professionnels principaux ; blanchisseurs ouvriers de 1re catégorie.

Sous-groupe 6. - Ouvriers professionnels ; blanchisseurs ouvriers professionnels.

Sous-groupe 7. - Photographes de 2e catégorie ; photographes adjoints.
