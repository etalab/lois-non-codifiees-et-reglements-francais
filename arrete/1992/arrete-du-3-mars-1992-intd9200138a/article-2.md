# Article 2

Les catégories d'informations nominatives enregistrées au fichier de gestion des demandes d'exonération de la taxe d'apprentissage sont les suivantes :

- numéro du dossier ;

- qualité ;

- nom, prénom ou raison sociale ;

- n° Siren ;

- code A.P.E. ;

- adresse ;

- n° et rue ;

- localité ;

- code postal ;

- bureau distributeur ;

- téléphone ;

- nombre de salariés ;

- date de création ;

- date d'arrêt ;

- motif d'arrêt.
