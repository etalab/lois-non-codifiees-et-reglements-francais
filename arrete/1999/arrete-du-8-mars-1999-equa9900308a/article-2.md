# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

Nom ;

Prénom ;

Civilité ;

Sexe ;

Date de naissance ;

Age ;

Adresse ;

Nombre d'heures de vol ;

Nom de l'employeur ;

Fonctions du candidat ;

Date de la visite ;

Diagnostic médical ;

Nom du médecin agréé ;

Décision prise par le conseil.
