# Article 2

Les informations nominatives enregistrées sont les suivantes :

- identité des agents :

- nom ;

- prénom ;

- initiales ;

- coordonnées (adresse, téléphone, télécopie) ;

- environnement du poste informatique de travail :

- adresse réseau ;

- serveur par défaut ;

- login par défaut ;

- groupes d'appartenance ;

- pour chaque message sortant du MATE (durée de stockage limitée à 3 mois) :

- l'expéditeur ;

- le destinataire ;

- l'objet ;

- la date et l'heure.
