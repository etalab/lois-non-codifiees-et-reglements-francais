# Article 3

Les taux de la taxe à la première vente prévue à l'article 4 du décret susvisé sont fixés, par quintal, pour la campagne 2000-2001, ainsi qu'il suit :

8,54 F pour les semences certifiées de céréales hybrides ;

2,70 F pour les semences certifiées de céréales de paille ;

4,41 F pour les semences de base de céréales à paille ;

26,43 F pour les semences de maïs hybrides simples ;

20,69 F pour les semences de maïs hybrides 3 voies ;

16,28 F pour les semences de maïs hybrides doubles ;

29,50 F pour les semences de sorgho ;

10,46 F pour les semences certifiées de vesces, féveroles et sainfoin ;

6,50 F pour les semences certifiées de lupins, pois protéagineux et autres pois fourragers ;

42,35 F pour les semences de graminées fourragères et autres semences fourragères en espèces pures ;

57,11 F pour les semences de graminées fourragères et autres semences fourragères en mélanges ;

67,92 F pour les semences de betteraves sucrières ;

14,08 F pour les semences de betteraves fourragères ;

41,76 F pour les semences de chicorée industrielle ;

8,60 F pour les plants de pommes de terre ;

25,19 F pour les semences de lin ;

58,38 F pour les semences de chanvre ;

26,28 F pour les semences de soja et de ricin ;

28,00 F pour les semences de moutarde ;

146,00 F pour les semences de tournesol ;

67,97 F pour les semences de colza hybride ;

63,46 F pour les autres semences oléagineuses ;

7,47 F pour les semences standards de pois, lentilles et fèves ;

9,47 F pour les semences standards de haricots ;

12,24 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,84 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est égal à 107 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et à 79 F par cent unités pour les semences de chicorée industrielle en unités de 100 000 graines.
