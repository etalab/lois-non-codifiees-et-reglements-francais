# Article 3

Les montants des taxes prévues à l'article 4 du décret n° 87-40 du 26 janvier 1987 susvisé sont fixées, par quintal, ainsi qu'il suit :

12,25 F pour les semences certifiées de céréales hybrides ;

1,65 F pour les semences certifiées de céréales à paille ;

2,94 F pour les semences de base de céréales à paille ;

19,30 F pour les semences de maïs hybrides simples ;

15,00 F pour les semences de maïs hybrides trois voies ;

11,72 F pour les semences de maïs hybrides doubles ;

19,75 F pour les semences de sorgho ;

5,70 F pour les semences certifiées de vesces, féveroles et sainfoin ;

7,20 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

23,05 F pour les semences de graminées fourragères ;

23,05 F pour les autres semences fourragères ;

49,40 F pour les semences de betteraves sucrières ;

8,70 F pour les semences de betteraves fourragères ;

29,40 F pour les semences de chicorée industrielle ;

6,60 F pour les plants de pommes de terre ;

15,85 F pour les semences de lin ;

43,40 F pour les semences de chanvre ;

17,85 F pour les semences de soja ;

35,10 F pour les semences de moutarde commerciale ;

71,25 F pour les semences de tournesol hybride ;

47,60 F pour les semences de tournesol population ;

42,35 F pour les autres semences oléagineuses ;

0,51 F pour les semences standard de pois, lentilles et fèves ;

0,51 F pour les semences standard de haricots ;

0,51 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,14 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est au plus égal à 75,20 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères, présentées en unités de 100 000 graines, et à 53,85 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
