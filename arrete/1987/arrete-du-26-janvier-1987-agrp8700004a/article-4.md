# Article 4

Le montant de la taxe additionnelle prévue à l'article 5 du décret n° 87-40 du 26 janvier 1987 susvisé est fixé, par quintal, ainsi qu'il suit :

0,35 F pour les semences de céréales ;

0,44 F pour les semences de maïs ;

2,21 F pour les semences de sorgho ;

2,11 F pour les semences certifiées de vesces, féveroles et sainfoin ;

1,77 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

8,50 F pour les autres semences fourragères ;

1,83 F pour les semences de betteraves et de chicorée industrielle ;

0,10 F pour les plants de pommes de terre ;

2,90 F pour les semences de lin ;

2,11 F pour les semences de soja ;

40,75 F pour les semences de tournesol ;

5,27 F pour les semences de moutarde commerciale ;

5,27 F pour les autres semences oléagineuses ;

5,09 F pour les semences standard de pois potagers, fèves et lentilles ;

6,60 F pour les semences standard de haricots ;

8,70 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles.

Le taux est égal à 5,70 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères et de chicorée industrielle présentées en unités de 100 000 graines.
