# Article 2

Les informations accessibles en consultation par le système L.I.C.R.E. sont les suivantes :

- le nom usuel et le prénom ;

- l'adresse ;

- la date de naissance ;

- le sexe ;

- le numéro de sécurité sociale (à titre d'identifiant) ;

- le code de l'emploi recherché ;

- le type de l'allocation attribuée ;

- le nombre de jours indemnisés ;

- l'interruption d'indemnisation pour maladie ;

- le nombre de jours restant à indemniser dans le type d'allocation versée ou la durée du droit initialement ouvert ;

- les admissions successives ;

- la date d'ouverture des droits ;

- la date de la dernière inscription ;

- l'indication de la dispense de recherche d'emploi ;

- le montant journalier de l'allocation en francs.

Les directions départementales du travail et de l'emploi pourront porter dans le système L.I.C.R.E. les résultats du contrôle effectué.
