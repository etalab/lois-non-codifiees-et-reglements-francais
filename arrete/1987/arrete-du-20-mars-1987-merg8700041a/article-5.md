# Article 5

Le rôle d'équipage comporte les énonciations suivantes [*contenu*]:

- en ce qui concerne le navire :

- le nom ;

- le numéro et le port d'immatriculation ;

- les caractéristiques (tonnage, longueur, puissance de l'appareil propulsif, lieu et époque de construction) ;

- le nom du propriétaire ;

- le nom de l'armateur ;

- le port d'armement ;

- le genre de navigation pratiquée ;

- en ce qui concerne chaque membre du personnel embarqué :

- les nom, prénoms et numéro d'identification ;

- la fonction exercée à bord ;

- le titre de qualification permettant l'exercice de cette fonction ;

- la date d'embarquement et la date de débarquement.
