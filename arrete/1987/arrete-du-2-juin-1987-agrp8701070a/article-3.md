# Article 3

Les taux des taxes prévues à l'article 4 du décret n° 87-40 du 26 janvier 1987 susvisé sont fixés, par quintal, ainsi qu'il suit :

12,45 F pour les semences certifiées de céréales hybrides ;

1,68 F pour les semences certifiées de céréales à paille ;

3,00 F pour les semences de base de céréales à paille ;

19,63 F pour les semences de maïs hybrides simples ;

15,25 F pour les semences de maïs hybrides 3 voies ;

11,92 F pour les semences de maïs hybrides doubles ;

20,10 F pour les semences de sorgho ;

5,80 F pour les semences certifiées de vesces, féveroles et sainfoin ;

7,20 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

23,45 F pour les semences de graminées fourragères ;

23,45 F pour les autres semences fourragères ;

50,25 F pour les semences de betteraves sucrières ;

8,85 F pour les semences de betteraves fourragères ;

29,90 F pour les semences de chicorée industrielle ;

6,70 F pour les plants de pommes de terre ;

16,12 F pour les semences de lin ;

44,15 F pour les semences de chanvre ;

18,15 F pour les semences de soja ;

35,70 F pour les semences de moutarde commerciale ;

72,45 F pour les semences de tournesol hybride ;

48,40 F pour les semences de tournesol population ;

43,07 F pour les autres semences oléagineuses ;

0,52 F pour les semences standard de pois, lentilles et fèves ;

0,52 F pour les semences standard de haricots ;

0,52 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,18 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est égal à 76,48 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines, et à 54,77 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
