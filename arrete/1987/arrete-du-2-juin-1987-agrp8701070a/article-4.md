# Article 4

Le taux de la taxe additionnelle prévue à l'article 5 du décret n° 87-40 du 26 janvier 1987 susvisé est fixé, par quintal, ainsi qu'il suit :

0,36 F pour les semences de céréales ;

0,45 F pour les semences de maïs ;

2,25 F pour les semences de sorgho ;

2,15 F pour les semences certifiées de vesces, féveroles et sainfoin ;

1,80 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

8,65 F pour les semences fourragères ;

1,85 F pour les semences de betteraves et de chicorée industrielle ;

0,10 F pour les plants de pommes de terre ;

2,95 F pour les semences de lin ;

2,15 F pour les semences de soja ;

41,45 F pour les semences de tournesol ;

5,35 F pour les semences de moutarde commerciale ;

5,18 F pour les semences standard de pois potagers, fèves et lentilles ;

6,70 F pour les semences standard de haricots ;

8,85 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

Le taux est égal à 5,80 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères et de chicorée industrielle présentées en unités de 100 000 graines.
