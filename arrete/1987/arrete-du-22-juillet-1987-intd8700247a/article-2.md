# Article 2

Les catégories d'informations nominatives, enregistrées dans le fichier de gestion des expulsions locatives, sont les suivantes :

1° Concernant le locataire :

- nom ;

- prénoms ;

- sexe ;

- date de naissance ;

- numéro d'allocations familiales ;

- description de la situation familiale et professionnelle ;

- description du logement ;

- date et motif du jugement d'expulsion ;

- date et durée du jugement de tutelle.

2° Concernant le propriétaire :

- nom ;

- adresse ;

- nom de l'huissier agissant au nom du propriétaire ;

- date de réquisition de la force publique.

3° Concernant les fonctionnaires de la sous-préfecture :

- nom.
