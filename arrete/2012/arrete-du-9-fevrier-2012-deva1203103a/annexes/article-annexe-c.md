# Article Annexe C

DONNÉES MENSUELLES TRANSMISES POUR LES VOLS AUTRES QUE CEUX DE TRANSPORT PUBLIC

Aérodrome source (code OACI).

Nombre de mouvements mensuels et répartition pour les items suivants :

- nature du vol : local ou voyage ;

- type de mouvement : départ, arrivée (pour les vols voyages) ;

- Type d'exploitant : aéroclubs, privés français, privés étrangers, travail aérien, appareils commerciaux sans passager, autres vols (officiels, etc) ;

- Type de motorisation avion : moteurs à pistons, turbopropulseurs, réacteurs, hélicoptères, planeurs.
