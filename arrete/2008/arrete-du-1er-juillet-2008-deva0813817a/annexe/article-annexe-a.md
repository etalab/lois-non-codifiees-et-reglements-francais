# Article Annexe A

**DONNÉES PAR VOL TRANSMISES POUR LES VOLS DE TRANSPORT AÉRIEN PUBLIC**

Aérodrome où a lieu le vol, dénommé aérodrome source, identifié par son code OACI ;

Transporteur aérien exploitant identifié par son code OACI ou son code IATA, ou à défaut par son nom ;

En cas d'affrètement : identification du transporteur aérien affréteur, identifié par son code OACI ou son code IATA ;

En cas de vol franchisé : identification du transporteur aérien franchiseur, identifié par son code OACI ou son code IATA ;

Transporteurs associés au vol, identifiés par leur code OACI, ou leur code IATA, ou à défaut par leur nom ;

Identification des vols assurés par un service routier (fret camionné) ;

Nature du vol : régulier, non régulier ;

Identification des vols charter selon codification IATA ;

Classification du vol : mixte, cargo, postal, autre ;

Numéro de vol ;

Circonstance du vol : vol en déroutement (oui/non) ;

Type du mouvement : vol au départ ou à l'arrivée ;

Premier aérodrome (vol au départ) ou dernier aérodrome (vol à l'arrivée) escale du vol, identifié par son code OACI ;

Nombre de passagers totaux (total du vol) ;

Nombre de passagers de moins de deux ans ;

Nombre de passagers en transit ;

Fret total transporté sur le vol : total en kg ;

Poste transportée sur le vol : total en kg.

Pour les escales successives identifiées du vol, seront fournies les données suivantes :

Nom de l'escale identifiée par son code OACI ;

Nombre de passagers locaux à cette escale ;

Nombre et poids (kg) des bagages à cette escale ;

Fret direct (kg) à cette escale ;

Poste directe (kg) à cette escale ;

Passagers en correspondance arrivant et partant en moins de 24 heures sur un aéronef différent, ou sur un même aéronef portant des numéros de vol différents, la destination finale de son voyage étant différente de son origine ;

Nombre de sièges passagers offerts à la vente sur le vol au sens du règlement (CE) n° 437/2003 du 27 février 2003 ;

Immatriculation de l'aéronef ;

Type d'aéronef : code OACI ;

Date et horaire auxquels l'aéronef quitte ou atteint l'aire de stationnement (*) (année, mois, jour ; heure, minute en heure locale ou universelle) ;

Date et horaire auxquels l'aéronef a décollé de la piste ou atterri sur la piste (*) (année, mois, jour ; heure, minute en heure locale ou universelle) ;

Date et horaire auxquels il était prévu que l'aéronef quitte ou atteigne l'aire de stationnement (année, mois, jour ; heure, minute en heure locale ou universelle) ;

Cause (s) de retard du vol : code IATA de chaque cause identifiée et la durée du retard imputé ;

Indicatif navigation aérienne du vol (*) ;

Piste utilisée : orientation ou QFU (*) ;

Régime de vol : IFR/VFR (*) ;

Balisage : en service ou non (*) ;

Visibilité horizontale (*) ;

Aérogare utilisée ;

Terminal utilisé.

<font color="#808080">
<em>(*) Selon éléments transmis par les services de la navigation aérienne lorsqu'il existe un service de contrôle. </em>
</font>
