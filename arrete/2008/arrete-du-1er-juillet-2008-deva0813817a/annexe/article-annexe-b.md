# Article Annexe B

**DONNÉES PAR VOL TRANSMISES PAR LES EXPLOITANTS D'AÉRODROMES POUR LES VOLS AUTRES QUE CEUX DE TRANSPORT AÉRIEN PUBLIC**

Aérodrome où a lieu le mouvement d'aéronef considéré, identifié par son code OACI ;

Type du mouvement : vol au départ ou à l'arrivée ;

Circonstance du vol : vol en déroutement (oui/non) ;

Date et horaire auxquels l'aéronef a décollé de la piste ou atterri sur la piste (*) (année, mois, jour ; heure, minute en heure locale ou universelle) ;

Date et horaire auxquels l'aéronef quitte ou atteint l'aire de stationnement (*) (année, mois, jour ; heure, minute en heure locale ou universelle) ;

Immatriculation de l'aéronef ;

Type d'aéronef code OACI ;

Nature du vol : local ou voyage ;

Pour un vol local :

Nombre de touchées avec heures du début et de fin du vol.

Pour un voyage :

Aérodrome de destination (escale suivante) ou d'origine (escale précédente) du vol identifié par son code OACI ;

Nombre de passagers à bord ;

Indicatif navigation aérienne du vol (*) ;

Piste utilisée : orientation ou QFU (*) ;

Régime de vol : IFR ou VFR (*) ;

Balisage : en service ou non (*) ;

Visibilité horizontale (*) ;

<font color="#808080">
<em>(*) Selon éléments transmis par les services de la navigation aérienne lorsqu'il existe un service de contrôle. </em>
</font>
