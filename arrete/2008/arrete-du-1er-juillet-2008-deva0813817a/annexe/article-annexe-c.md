# Article Annexe C

**FORMULAIRE MENSUEL DES MOUVEMENTS TRANSMIS PAR LES EXPLOITANTS D'AÉRODROME**

Aérodrome identifié par son code OACI ;

Nombre de mouvements pour les items suivants :

Nature du vol : local ou voyage ;

Vols d'aéroclubs ;

Vols privés français ;

Vols privés étrangers ;

Vols de travail aérien ;

Vols d'appareils commerciaux (sans passagers) ;

Autres vols : officiels, etc.

Pour les vols voyages : vol au départ ou à l'arrivée.

Types de motorisations avions :

Moteurs à pistons ;

Turbopropulseurs ;

Réacteurs ;

Hélicoptères ;

Planeurs.
