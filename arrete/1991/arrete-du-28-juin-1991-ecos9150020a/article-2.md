# Article 2

Les informations individuelles enregistrées sont :

- numéro Siren de l'entreprise ;

- code d'activité principale exercée (A.P.E.) ;

- chiffre d'affaires à l'exportation (CAEX) ;

- pourcentage ou partie du CAEX facturé en dollars U.S. ;

- code permettant de repérer les " groupeurs " à l'export ;

- ventilation du CAEX selon quinze destinations :

- dix pays de la C.E.E. (U.E.B.L., Pays-Bas, R.F.A., Italie, Royaume-Uni, Irlande, Danemark, Grèce, Portugal, Espagne) ;

- Etats-Unis ;

- Japon ;

- A.E.L.E. ;

- O.C.D.E. ;

- Autres ;

- total des importations ;

- code permettant de repérer les " groupeurs " à l'import ;

- ventilation des importations selon le même découpage par pays d'origine.
