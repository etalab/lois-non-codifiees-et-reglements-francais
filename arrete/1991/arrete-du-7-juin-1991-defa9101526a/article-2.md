# Article 2

Les catégories d'informations traitées sont les suivantes :

- identité : nom, code Siret ou Siren, code A.P.E., adresse, téléphone, télex, principaux actionnaires ;

- activité professionnelle : forme juridique, code produits, catégorie économique ;

- situation économique et financière : code banque, numéro R.I.B., chiffre d'affaires, indicateur de dépendance ;

- service acheteurs : référence du rédacteur, dates de rédaction, renseignements confidentiels industrie ;

- contrôle qualité : code qualité, RAQ, code homologation, SIAR responsable.
