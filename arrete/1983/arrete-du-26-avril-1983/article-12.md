# Article 12

Les agents visés à l'article 11 ci-dessus peuvent demander leur inscription sur la liste d'aptitude dans les conditions suivantes :

Groupe 1 : classe(s) C 2 ou (et) C 1 ;

Groupe 2 : classe(s) C 2 ou (et) C 1 ; classe B 1. Toutefois, une inscription dans cette classe ne peut être demandée que conjointement à une inscription dans l'une ou les deux classes ci-dessus.
