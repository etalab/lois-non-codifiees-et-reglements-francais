# Article 2

Les estampilles comprennent :

Une estampille plaquette pour l'exportation ;

Une estampille adhésive pour l'exportation ;

Une estampille adhésive nationale.

La reproduction de l'estampille pour l'exportation ou de l'estampille nationale peut être imprimée sur les enveloppes de conditionnement. L'estampille pour l'exportation peut être reproduite sur une étiquette commerciale.

Toute autre modalité d'estampillage pourra, éventuellement être autorisée pour satisfaire les exigences des pays importateurs.
