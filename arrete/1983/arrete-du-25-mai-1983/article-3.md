# Article 3

Les informations traitées sont les suivantes :

Titre (N.MM.ML.) ;

Nom et prénom ou raison sociale du livreur ;

Adresse légale du livreur (numéro de voirie et rue, commune, code postal, bureau distributeur) ;

Numéro spécifique et code à usage interne délivrés par l'Office national interprofessionnel des céréales ;

Numéro spécifique à usage interne délivré au collecteur par l'Office national interprofessionnel des céréales ;

Commune dans laquelle s'exerce l'activité du collecteur ;

Quantités livrées par produit céréalier soumis aux taxes complémentaires.
