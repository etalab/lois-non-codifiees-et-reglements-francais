# Article 4

Le montant des primes convertibles en bonifications d'intérêt visées à l'article 56 du décret du 24 janvier 1972 susvisé, pour la construction de logements-foyers à usage locatif, est fixé forfaitairement par type de logement, conformément au tableau ci-dessous:

Jusqu'au 30 juin 1985

TYPE DE LOGEMENTS : I

- Pour une personne

PRIMES POUR LA CONSTRUCTION DE LOGEMENTS :

Pour les personnes âgées (isolées ou couples) (en francs) :

64.589

Pour les personnes seules (autres que les personnes âgées) (en francs) : 50.210

TYPE DE LOGEMENTS : I

- Pour deux personnes

PRIMES POUR LA CONSTRUCTION DE LOGEMENTS :

Pour les personnes âgées (isolées ou couples) (en francs) :

80.077

Pour les personnes seules (autres que les personnes âgées) (en francs) : 61.264

TYPE DE LOGEMENTS : I

- Pour trois personnes

PRIMES POUR LA CONSTRUCTION DE LOGEMENTS :

Pour les personnes âgées (isolées ou couples) (en francs) :

Pour les personnes seules (autres que les personnes âgées) (en francs) : 76.826

TYPE DE LOGEMENTS : I bis

PRIMES POUR LA CONSTRUCTION DE LOGEMENTS :

Pour les personnes âgées (isolées ou couples) (en francs) :

Pour les personnes seules (autres que les personnes âgées) (en francs) : 89.800
