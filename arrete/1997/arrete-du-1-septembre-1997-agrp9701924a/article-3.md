# Article 3

Les taux de la taxe à la première vente prévue à l'article 4 du décret du 28 mars 1996 susvisé sont fixés, par quintal, pour la campagne 1997-1998, ainsi qu'il suit :

16,15 F pour les semences certifiées de céréales hybrides ;

2,58 F pour les semences certifiées de céréales à paille ;

4,22 F pour les semences de base de céréales à paille ;

25,26 F pour les semences de maïs hybrides simples ;

19,78 F pour les semences de maïs hybrides 3 voies ;

15,56 F pour les semences de maïs hybrides doubles ;

28,22 F pour les semences de sorgho ;

10,00 F pour les semences certifiées de vesces, féveroles et sainfoin ;

8,32 F pour les semences certifiées de lupins, pois protéagineux et autres pois fourragers ;

40,50 F pour les semences de graminées fourragères et autres semences fourragères en espèces pures ;

54,63 F pour les semences de graminées fourragères et autres semences fourragères en mélanges ;

64,90 F pour les semences de betteraves sucrières ;

13,50 F pour les semences de betteraves fourragères ;

40,00 F pour les semences de chicorée industrielle ;

8,20 F pour les plants de pommes de terre ;

24,09 F pour les semences de lin ;

55,85 F pour les semences de chanvre ;

25,13 F pour les semences de soja et de ricin ;

51,05 F pour les semences de moutarde commerciale ;

139,65 F pour les semences de tournesol hybride ;

65,00 F pour les semences de colza hybrides ;

60,70 F pour les autres semences oléagineuses ;

7,14 F pour les semences standards de pois, lentilles et fèves ;

9,05 F pour les semences standards de haricots ;

11,70 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,72 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est égal à 103 F par 100 unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et à 76 F par 100 unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
