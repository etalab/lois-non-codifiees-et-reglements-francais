# Article 2

Les informations nominatives enregistrées dans ce fichier sont les suivantes, quel que soit le résultat de l'examen :

- nom ;

- prénom ;

- date et lieu de naissance ;

- sexe ;

- adresse du domicile ;

- numéro de téléphone ;

- courrier ;

- diplôme(s) préparé(s) ;

- diplôme(s) obtenu(s) ;

- date(s) d'obtention ;

- direction régionale ayant délivré le(s) diplôme(s).
