# Article 2

Les informations du document d'accompagnement traitées concernent :

- les données relatives à l'identification de l'envoi : code d'identification du produit, pays de destination, pays de destination finale, date d'expédition ;

- les données relatives à l'identification de l'opérateur : numéro de TVA, type d'opérateur

Les statistiques obtenues et diffusées par la douane sont des résultats croisés : produits/pays et pays/produits, présentés par types d'opérateurs.
