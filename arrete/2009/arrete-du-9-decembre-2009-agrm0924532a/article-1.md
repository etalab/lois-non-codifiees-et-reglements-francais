# Article 1

Les débarquements des navires utilisant les filets fixes dans les zones CIEM II a, IV a, V b, VII b, c, j k, VIII, IX, X et XII ne peuvent être effectués que dans les ports maritimes listés ci-dessous :

Boulogne-sur-Mer ;

Brest ;

Douarnenez ;

Saint-Guénolé ;

Le Guilvinec ;

Concarneau ;

Lorient ;

Noirmoutier ;

Yeu ;

Les Sables-d'Olonne ;

Saint-Jean-de-Luz.
