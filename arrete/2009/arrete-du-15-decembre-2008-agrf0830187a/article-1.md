# Article 1

Des concessions foncières peuvent être accordées à des agriculteurs sur les territoires des communes dont la liste suit, pour y pratiquer la culture sur abattis :

Saint-Laurent ;

Awala-Yalimapo ;

Apatou ;

Grand-Santi ;

Papaïchton ;

Maripasoula ;

Camopi ;

Saint-Georges ;

Regina ;

Ouanary ;

Saul ;

Saint-Elie ;

Iracoubo ;

Mana ;

Roura ;

Sinnamary.
