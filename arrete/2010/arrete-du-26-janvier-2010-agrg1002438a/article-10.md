# Article 10

Dans le cadre de l'identification électronique des animaux de l'espèce bovine, le ministre chargé de l'agriculture délivre un agrément provisoire aux repères auriculaires suivants :

Pour la société Reyflex :

- le modèle " ReyFID COMBO FDX ", de code d'agrément " FR 40 " ;

- le modèle " ReyFID COMBO HDX ", de code d'agrément " FR 23 ".

Pour la société Allflex :

- le modèle " SELECT ULTRA HDX ", de code d'agrément " FR 43 " ;

- le modèle " SELECT ULTRA FDX ", de code d'agrément " FR 44 " ;

Pour la société Chevillot :

- le modèle " AXAPLUS HDX ", de code d'agrément " FR 45 " ;

- le modèle " AXAPLUS FDX ", de code d'agrément " FR 46 "
