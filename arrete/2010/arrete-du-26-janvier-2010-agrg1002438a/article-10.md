# Article 10

Dans le cadre de l'identification électronique des animaux de l'espèce bovine, le ministre chargé de l'agriculture délivre, pour la mise en œuvre des projets qu'il a approuvés, pour les animaux des exploitations qui participent aux tests sur les repères auriculaires de type pendentif/pendentif électronique et dans la limite des besoins pour lesdits tests dans les départements pilotes, un agrément transitoire comme second marquage tel que prévu dans le règlement (CE) n° 911/2004 susvisé, aux repères auriculaires suivants :

Pour la société Reyflex :

― le modèle « ReyFID COMBO », de code d'agrément « FR 40 ».

Pour la société Allflex :

― le modèle « SELECT ULTRA HDX », de code d'agrément « FR 43 ».
