# Article 2

Dans le cadre de l'identification des animaux des espèces ovine et caprine, le ministre chargé de l'agriculture agrée, pour l'identification des animaux de ces espèces destinés à être abattus avant l'âge de deux mois, les repères auriculaires de type barrette rigide suivants :

― le modèle « Tiptag », de la société Chevillot, de code d'agrément « FR 50 » ;

― le modèle « Ovibag », de la société Néodis, de code d'agrément « FR 51 » ;

― le modèle « Reytag », de la société Reyflex, de code d'agrément « FR 52 ».
