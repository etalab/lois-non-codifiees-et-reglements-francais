# Article 3

Dans le cadre de l'identification des animaux des espèces ovine et caprine, le ministre chargé de l'agriculture agrée, pour l'identification de tous les animaux de ces espèces, les repères auriculaires conventionnels de type pendentif suivants :

Pour la société Reyflex :

- la combinaison suivante : partie femelle "ReyVis S" de code d'agrément "FR 80" et partie mâle "ReyVIS S"de code d'agrément "FR 80" ;

― la combinaison suivante : partie femelle  Supermini  de code d'agrément  FR 85  et partie mâle  Supermini  de code d'agrément  FR 85  ;

― la combinaison suivante : partie femelle  Supermini  de code d'agrément  FR 85  et partie mâle  MPLC  de code d'agrément  FR 83  ;

― la combinaison suivante : partie femelle  Supermini  de code d'agrément  FR 85  et partie mâle  MG  de code d'agrément  FR 84 .

Pour la société Néodis :

― la combinaison suivante : partie femelle  Oviflex  de code d'agrément  FR 87  et partie mâle  Oviflex  de code d'agrément  FR 87 .

Pour la société Chevillot :

― la combinaison suivante : partie femelle  Axaflex Ovina  de code d'agrément  FR 91  et partie mâle  Axaflex Ovina  de code d'agrément  FR 91  ;

― la combinaison suivante : partie femelle  Axaflex Couple 16  de code d'agrément  FR 92  et partie mâle  Axaflex Couple 16  de code d'agrément  FR 92  ;

― la combinaison suivante : partie femelle  Axaflex Couple 14  de code d'agrément  FR 94  et partie mâle  Axaflex Couple 14  de code d'agrément  FR 94 .

Pour la société Allflex :

― la combinaison suivante : partie femelle  Sheep Ultra  de code d'agrément  FR 95  et partie mâle  Junior  de code d'agrément  FR 97  ;

― la combinaison suivante : partie femelle  Sheep Ultra  de code d'agrément  FR 95  et partie mâle  Senior  de code d'agrément  FR 96 .

Pour la société Agid :

― la combinaison suivante : partie femelle  Classic tag  de code d'agrément  FR 93  et partie mâle  Classic tag  de code d'agrément  FR 93 .

Pour la société OS ID, distribuée par la société Allflex :
