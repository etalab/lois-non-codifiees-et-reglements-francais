# Article 7 bis

Dans le cadre de l'identification visuelle des animaux de l'espèce caprine, le ministre chargé de l'agriculture agrée, pour tous les animaux de cette espèce, la bague de paturon suivante :

- le modèle "Capritag" de la société Chevillot, de code d'agrément "FR 56" ;

- le modèle " RayTrace visuel V3 " de la société ARaymondlife, de code d'agrément " FR 57 " ;

- le modèle " PATUFLEX-visuel version 3 " de la société Reyflex, de code d'agrément " FR 58 ".
