# Article 5

Dans le cadre de l'identification électronique des animaux des espèces ovine et caprine, le ministre chargé de l'agriculture agrée, pour l'identification des animaux de ces espèces, les repères auriculaires de type bouton/pendentif électroniques suivants :

Pour la société Chevillot :

― la combinaison suivante : partie femelle  Sheep air tag  de code d'agrément  FR 75  et partie mâle  Axaflex Ovina  de code d'agrément  FR 91  ;

- la combinaison suivante : partie femelle "Sheep Air Tag" de code d'agrément "FR 75" et partie mâle "Axaflex couple 16" de code d'agrément "FR 92" ;

- la combinaison suivante : partie femelle "Sheep Air Tag" de code d'agrément "FR 75" et partie mâle "Axaflex couple 14" de code d'agrément "FR 94" ;

- la combinaison suivante : partie femelle "Sheep Air Tag" de code d'agrément "FR 75" et partie mâle "Junior" de code d'agrément "FR 97".

Pour la société Reyflex :

― la combinaison suivante : partie femelle  bouton FRCE FDX  de code d'agrément  FR 72  et partie mâle  Reyflex Mini  de code d'agrément  FR 85  ;

Pour la société Néodis :

― la combinaison suivante : partie femelle  bouton FDX - Ovitronic  de code d'agrément  FR 74  et partie mâle  Oviflex  de code d'agrément  FR 87  ;

- la combinaison suivante : partie femelle "ReyFID Button S" de code d'agrément "FR 77" et partie mâle "ReyVIS S" de code d'agrément "FR 77".

Pour la société OS ID distribuée par la société Allflex :

― la combinaison suivante : partie femelle  Combi E23  de code d'agrément  FR 76  et partie mâle  Combi Mini 2000  de code d'agrément  FR 98 .
