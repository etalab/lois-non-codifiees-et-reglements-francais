# Article 8

Dans le cadre de l'identification des animaux de l'espèce bovine, le ministre chargé de l'agriculture agrée, pour l'identification de tous les animaux de cette espèce, les repères auriculaires de type pendentif-pendentif conventionnel suivants :

― le modèle " Axaflex ", de la société Chevillot, de code d'agrément " FR 15 " ;

― le modèle " Allflex ", de la société Allflex Europe, de code d'agrément " FR 18 " ;

― le modèle " Ultra ", de la société Allflex Europe, de code d'agrément " FR 27 " ;

― le modèle " Reyflex ", de la société Reyflex, de code d'agrément " FR 29 " ;

― le modèle " Zeetag Z 3 laser " de la société Zeetag, de code d'agrément " FR 30 " ;

- le modèle " Bov'ardes " de la société Ardes, de code d'agrément " FR 31 " ;

- le modèle " Ecoflex " de la société Chevillot, de code d'agrément " FR 47 ".
