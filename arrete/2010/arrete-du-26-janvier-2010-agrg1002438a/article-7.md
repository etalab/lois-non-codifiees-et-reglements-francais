# Article 7

Dans le cadre de l'identification électronique des animaux de l'espèce caprine, le ministre chargé de l'agriculture agrée, pour l'identification de tous les animaux de cette espèce, les bagues de paturon suivantes :

- le modèle " RayTrace V3 " de la société ARaymondlife, de code d'agrément " FR 54 ";

- le modèle CAPRITAG , de la société Chevillot, de code d'agrément " FR 55 " ;

- le modèle "PATUFLEX - version 3" de la société Reyflex de code d'agrément " FR 53 ".
