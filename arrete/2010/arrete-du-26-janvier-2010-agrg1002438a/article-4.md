# Article 4

Dans le cadre de l'identification des animaux des espèces ovine et caprine, le ministre chargé de l'agriculture agrée, pour l'identification de tous les animaux de ces espèces, les repères auriculaires de type barrette souple suivants :

- le modèle "Strip tag ", de la société AGID, de code d'agrément " FR 64 " ;

- le modèle " Cheviflex ", de la société Chevillot, de code d'agrément " FR 60 " ;

- le modèle " Fba/Mba ", de la société Reyflex, de code d'agrément " FR 62 " ;

- le modèle " Orimark ", de la société Néodis, de code d'agrément " FR 61 " ;

- le modèle " RayTrace ", de la société A. Raymondlife, de code d'agrément " FR 63 ".
