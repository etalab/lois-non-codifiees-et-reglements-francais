# Article 3

Les insignes de grade des agents techniques de l'environnement sont ainsi définis :

Les galons sont portés sur des pattes d'épaule de couleur grise selon les modalités suivantes :

― agent technique stagiaire : un galon argent en V renversé ;

― agent technique : deux galons argent en V renversé ;

― agent technique principal de 2e classe : trois galons argent en V renversé ;

― agent technique principal de 1re classe : un galon trait argent avec un fil de soie gris foncé.

Les insignes portés sur le képi par les agents techniques stagiaires, les agents techniques et les agents techniques principaux de 2e et de 1re classe sont les suivants : fausse jugulaire argent et soutache argent à fil gris.
