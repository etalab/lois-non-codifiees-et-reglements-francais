# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

1° Concernant le requérant :

- nom ou raison sociale ;

- prénom ;

- adresse ;

- avocat (le cas échéant) ;

- description de l'affaire ;

- état d'instruction du dossier.

2° Concernant le défendeur :

- nom ou raison sociale ;

- prénom ;

- adresse ;

- avocat (le cas échéant).

3° Concernant les experts intervenant dans l'instruction des dossiers :

- nom, prénom, année de naissance ;

- diplômes ;

- spécialités ;

- tarif ;

- nombre d'expertises ;

- nombre de rappels.

4° Concernant les fonctionnaires du tribunal administratif :

- nom des membres du tribunal administratif et du greffier intervenant dans le traitement de chaque affaire.
