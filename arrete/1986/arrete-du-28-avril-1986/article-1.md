# Article 1

Les diplômes qui peuvent être préparés dans les conditions définies à l'article R. 117-7-1 du code du travail sont :

1. Les sous-options de l'option du certificat d'aptitude professionnelle agricole (C.A.P.A.) détenu par le candidat, autres que celles dont il est titulaire ;

2. Les options de C.A.P.A. réputées connexes à l'option du C.A.P.A. initial possédé par le candidat, désignées ci-dessous ; le C.A.P. connexe doit être préparé après le C.A.P. initial.

C.A.P. AGRICOLE INITIAL :

Employé d'exploitation agricole de polyculture élevage.

C.A.P. AGRICOLE CONNEXE :

- sous-option A : vacher ;

- sous option D : éleveurs de chiens.

Employé d'élevage de petits animaux :

- sous-option B : aviculture ;

- sous-option C : gibier d'élevage.
