# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

Erreurs de caisse par bureau :

- nom du bureau ;

- montant des erreurs ;

- date des erreurs.

Erreurs de caisse par agent :

- nom de l'agent ;

- montant de l'erreur ;

- date de l'erreur ;

- cumul des erreurs.

Indemnité accordée :

- nom de l'agent ;

- montant de l'indemnité ;

- date de l'indemnité ;

- cumul des indemnités.
