# Article ANNEXE

PROGRAMME DES ENSEIGNEMENTS

I. - Administration publique

1. Les administrations :

économiques et financières ;

sociales ;

éducatives ;

techniques ;

de sécurité et de défense ;

territoriales.

2. Les institutions européennes :

les institutions communautaires ;

l'Acte unique européen ;

les politiques communautaires ;

les fonds structurels.

II. - Techniques juridiques

1. Droit public :

droit constitutionnel : la Constitution de 1958, l'organisation des pouvoirs, les rapports entre les pouvoirs ;

droit administratif et institutions administratives :

l'organisation administrative, la justice administrative, la réglementation juridique de l'activité administrative.

2. Les modes de gestion des services publics.

3. Les types d'interventions publiques.

4. Le droit de l'informatique.

III. - Techniques budgétaires, financières et comptables

1. Finances publiques :

la place des finances publiques dans la vie nationale ;

le budget de l'Etat :

le poids économique du budget de l'Etat ;

l'exécution du budget de l'Etat ;

le contrôle du budget de l'Etat ;

le Trésor public ;

les finances locales.

2. Techniques budgétaires et financières :

élaboration du budget de l'Etat, des collectivités territoriales et des établissements publics ;

les règles de la comptabilité publique ;

les procédés de contrôle.

3. Les techniques comptables :

comptabilité privée générale ;

analyse financière ;

comptabilité analytique.

IV. - Techniques de communication,

d'organisation et de gestion

1. Techniques de communication :

techniques d'expression orale, entretiens et conduite de réunion ;

techniques de rédaction ;

relations administration-administrés ;

relations informaticiens-utilisateurs d'informatique.

2. Techniques statistiques et d'enquêtes :

généralités sur les statistiques descriptives ;

les tableaux statistiques ;

les représentations graphiques ;

les caractéristiques de dispersion et de concentration ;

les papiers fonctionnels et les indices ;

le traitement des séries chronologiques ;

ajustement linéaire et corrélation simple ;

les techniques d'enquête.

3. Techniques d'organisation :

la pratique de l'organisation ;

nécessités et limites de l'organisation ;

les étapes d'une étude d'organisation.

4. La gestion des ressources humaines :

l'administration et la fonction ressources humaines ;

les règles applicables aux fonctionnaires ;

la gestion individuelle et la gestion collective ;

la gestion prévisionnelle ;

la gestion d'une cellule informatique.

5. Ergonomie des systèmes informatiques :

le poste de travail ;

les logiciels ;

l'organisation des tâches.

6. Sécurité des systèmes informatiques :

plan de sécurité ;

analyse des risques ;

contre-mesures pour le contrôle des accès logiques ;

responsabilité du gestionnaire de bases de données.

V. - Informatique

1. La programmation structurée.

2. Les matériels.

3. Eléments de politique informatique.

4. Elaboration d'un schéma directeur.

5. Méthode d'analyse : études théoriques et pratiques d'une méthode classique et d'une méthode systémique :

organisation du système d'information ;

techniques et outils au service de l'analyse ;

analyse organique.

6. Projets d'informatisation.

7. La structuration des données :

les tables ;

les fichiers ;

les bases de données ;

les progiciels de gestion.

8. Les systèmes :

différents types de systèmes ;

étude d'un système d'exploitation ;

aide à la conception, intelligence artificielle, système expert.

9. Les réseaux :

bases techniques ;

architecture de réseaux ;

gestion des réseaux ;

les réseaux locaux.

10. Bureautique :

les différents outils ;

liaison informatique-bureautique ;

les relations avec les utilisateurs.

11. Les contrats et marchés informatiques :

les règles applicables aux logiciels et aux matériels ;

élaboration d'un marché.
