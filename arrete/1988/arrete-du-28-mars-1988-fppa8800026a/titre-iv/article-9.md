# Article 9

Chacun des enseignements prévus à l'article 6 ci-dessus est sanctionné par une ou plusieurs épreuves de classement. La nature, le nombre, la durée et le coefficient des épreuves sont fixés comme suit :

administration publique : épreuve orale (durée : quinze minutes ; coefficient 2) ;

droit public ou finances publiques, au choix du jury : épreuve pratique (durée : trois heures ; coefficient 2) ;

techniques juridiques : rédaction d'une note administrative à partir d'un dossier ou d'un cas concret (durée : trois heures ; coefficient 2) ;

techniques budgétaires, financières et comptables :

rédaction d'une note administrative à partir d'un dossier de nature budgétaire ou financière (durée : trois heures ; coefficient 2) ;

comptabilité privée : épreuve pratique (durée : trois heures ; coefficient 2) ;

techniques de communication, d'organisation et de gestion :

statistiques : épreuve pratique (durée : trois heures ; coefficient 1) ;

organisation et gestion des ressources humaines : étude d'un cas pratique (durée : trois heures ; coefficient 1) ;

informatique :

analyse ; épreuve écrite théorique (durée : trois heures ; coefficient 2) ;

conduite de projets : deux projets d'informatisation menés en sous-groupes (coefficient 3 chacun) ;

bases de données : épreuve pratique (durée : quatre heures ; coefficient 2) ;

réseaux : épreuve écrite (durée : deux heures ; coefficient 1) ;

marchés informatiques : étude de cas (durée : trois heures ; coefficient 1) ;

épreuve orale devant le jury sur un sujet touchant à l'informatique dans l'administration, à partir d'un document, d'un texte, d'un thème et après une préparation de vingt minutes (durée :

vingt minutes ; coefficient 4) ;

stage :

note relative à la première période de stage (coefficient 2) ;

note relative à la deuxième période de stage (coefficient 4) ;

rapport de stage : épreuve orale (durée : 30 minutes ; coefficient 4).
