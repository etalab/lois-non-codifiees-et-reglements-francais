# Article 3

Les taux des taxes prévues à l'article 4 du décret n° 87-40 du 26 janvier 1987 susvisé sont fixés, par quintal, ainsi qu'il suit :

12,65 F pour les semences certifiées de céréales hybrides ;

1,71 F pour les semences certifiées de céréales à paille ;

3,05 F pour les semences de base de céréales à paille ;

19,95 F pour les semences de maïs hybrides simples ;

15,50 F pour les semences de maïs hybrides trois voies ;

12,12 F pour les semences de maïs hybrides doubles ;

20,45 F pour les semences de sorgho ;

5,90 F pour les semences certifiées de vesces, féveroles et sainfoin ;

7,30 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

23,85 F pour les semences de graminées fourragères ;

23,85 F pour les autres semences fourragères ;

51     F pour les semences de betteraves sucrières ;

9     F pour les semences de betteraves fourragères ;

30,40 F pour les semences de chicorée industrielle ;

6,70 F pour les plants de pommes de terre ;

16,40 F pour les semences de lin ;

45     F pour les semences de chanvre ;

18,45 F pour les semences de soja ;

36,30 F pour les semences de moutarde commerciale ;

73,70 F pour les semences de tournesol hybride ;

43,80 F pour les autres semences oléagineuses ;

0,53 F pour les semences standard de pois, lentilles et fèves ;

0,53 F pour les semences standard de haricots ;

0,53 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,22 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est au plus égal à 78 F par 100 unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et à 56 F par 100 unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
