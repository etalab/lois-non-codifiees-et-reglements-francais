# Article 4

Le taux de la taxe additionnelle prévue à l'article 5 du décret n° 87-40 du 26 janvier 1987 susvisé est fixé, par quintal, ainsi qu'il suit :

0,37 F pour les semences de céréales ;

0,46 F pour les semences de maïs ;

2,29 F pour les semences de sorgho ;

2,19 F pour les semences certifiées de vesces, féveroles et sainfoin ;

1,83 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

8,80 F pour les autres semences fourragères ;

1,88 F pour les semences de betteraves et de chicorée industrielle ;

0,10 F pour les plants de pommes de terre ;

3     F pour les semences de lin ;

2,19 F pour les semences de soja ;

42,15 F pour les semences de tournesol ;

5,44 F pour les semences de moutarde commerciale ;

5,44 F pour les autres semences oléagineuses ;

5,27 F pour les semences standard de pois potagers, fèves et lentilles ;

6,80 F pour les semences standard de haricots ;

9     F pour les semences certifiées de pois potagers, haricots, fèves et lentilles.

Le taux est égal à 6 F par 100 unités pour les semences de betteraves monogermes sucrières et fourragères et de chicorée industrielle présentées en unités de 100 000 graines.
