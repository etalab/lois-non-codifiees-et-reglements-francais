# Article ANNEXE II

Sur chaque emballage, les renseignements suivants devront être inscrits de façon indélébile :

Le nom du titulaire de l'homologation ;

Le numéro du lot (un même numéro de lot ne peut être attribué à une quantité supérieure à une tonne) ;

La mention : mélange microbilles de verre/granulats (80/20) ;

Le numéro d'homologation du mélange (ex. : MB.G.03 - MB.G.03H - MB.G.03T) :

- MB : microbilles de saupoudrage ;

- G : granulats ;

- 03 : numéro d'ordre ;

- H : microbilles hydrofugées ;

- T : microbilles traitées ;

- absence de lettre après le numéro d'ordre signifie :

" microbilles non hydrofugées ".
