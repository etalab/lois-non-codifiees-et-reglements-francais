# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

l'enseigne de commerce, le nom de commerçant, l'adresse, le numéro de téléphone, le type de commerce, les produits contrôlés, les prestations offertes et les prix relevés ;

le code Infraction et le code Nature infraction, le code Manquement aux cahiers des charges.
