# Article 2

Les catégories d'informations nominatives enregistrées sont les suivantes :

Concernant les bénéficiaires :

- nom ;

- prénom ;

- date de naissance ;

- numéro matricule au recrutement ;

- bureau du service national gestionnaire ;

- adresse de l'intéressé ;

- qualité, grade ;

- forme de service national ;

- affectation ;

- code objet affaire ;

- nature de la réponse.

Concernant les intervenants :

- nom ;

- prénom ;

- qualité, grade ;

- adresse ;

- numéro de l'intervention.

La durée de conservation des informations concernant les bénéficiaires et les intervenants est limitée à cinq ans à compter de la date de clôture de l'affaire.
