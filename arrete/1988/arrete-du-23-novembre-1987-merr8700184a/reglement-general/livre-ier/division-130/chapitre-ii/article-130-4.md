# Article 130.4

Implantation des centres de sécurité.

Un centre de sécurité des navires est implanté dans chacun des ports ci-dessous :

Dunkerque ;

Boulogne

Le Havre (Seine-Maritime Ouest) ;

Rouen (Seine-Maritime Est) ;

Caen ;

Saint-Malo ;

Brest ;

Concarneau ;

Lorient ;

Saint-Nazaire ;

La Rochelle ;

Bordeaux ;

Sète ;

Marseille ;

Fort-de-France ;

Le Port (île de La Réunion).
