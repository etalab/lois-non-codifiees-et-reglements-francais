# Article 1

Le prix maximum hors T.V.A. des analyses pratiquées sur les eaux destinées à l'alimentation, par les laboratoires agréés au titre du contrôle sanitaire des eaux, est fixé comme suit :

Analyse complète de type I :

- examen physico-chimique : 522,50 F ;

- examen bactériologique : 187,60 F ;

Analyse sommaire de type II : 364,60 F ;

Analyse de surveillance réduite de type III : 118,70 F.
