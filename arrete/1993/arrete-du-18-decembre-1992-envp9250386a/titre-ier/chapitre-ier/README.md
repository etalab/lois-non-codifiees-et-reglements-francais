# CHAPITRE Ier : Déchets admissibles       Les déchets admissibles dans les installations de stockage visées par le présent arrêté sont des déchets industriels spéciaux ultimes dont la définition est donnée à l'article 3    Ces déchets se répartissent en deux catégories :    A : déchets qui doivent être stabilisés dans un délai de deux ans ;    B : déchets qui doivent être stabilisés dans un délai de cinq ans.
