# Article 3

Les taux de la taxe à la première vente prévue à l'article 4 du décret du 18 février 1993 susvisé sont fixés, par quintal, ainsi qu'il suit :

14,65 F pour les semences certifiées de céréales hybrides ;

2,40 F pour les semences certifiées de céréales à paille ;

3,85 F pour les semences de base de céréales à paille ;

23,00 F pour les semences de maïs hybrides simples ;

18,10 F pour les semences de maïs hybrides trois voies ;

14,20 F pour les semences de maïs hybrides doubles ;

25,75 F pour les semences de sorgho ;

9,15 F pour les semences certifiées de vesces, féveroles et sainfoin ;

8,49 F pour les semences certifiées de lupins, pois protéagineux et autres pois fourragers ;

37,00 F pour les semences de graminées fourragères ;

37,00 F pour les autres semences fourragères ;

59,50 F pour les semences de betteraves sucrières ;

12,40 F pour les semences de betteraves fourragères ;

36,50 F pour les semences de chicorée industrielle ;

7,51 F pour les plants de pommes de terre ;

22,00 F pour les semences de lin ;

51,00 F pour les semences de chanvre ;

23,20 F pour les semences de soja ;

47,00 F pour les semences de moutarde commerciale ;

132,00 F pour les semences de tournesol hybride ;

55,46 F pour les semences de tournesol population ;

56,00 F pour les autres semences oléagineuses ;

6,55 F pour les semences standard de pois, lentilles et fèves ;

8,30 F pour les semences standard de haricots ;

10,70 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,50 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le taux est au plus égal à 95,50 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines et 69,70 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.
