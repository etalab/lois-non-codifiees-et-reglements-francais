# Article Annexe 3

1. Données relatives au prélèvement

A. - Caractéristiques de l'échantillon :

Type d'échantillon ;

Traitement de l'échantillon (par exemple : traitement chimique, délai de cinq jours, etc.).

B. - Date et heure :

Date de prélèvement de l'échantillon ;

Type de date (par exemple : date de début, date de fin, etc.) ;

Heure de prélèvement de l'échantillon (1) ;

Echelle de temps (1) (par exemple : GMT) ;

Durée du prélèvement (en heures).

C. - Lieu :

Nom de la localité ;

Code de la nomenclature des unités territoriales statistiques (NUTS). La liste des codes régionaux pour la France est disponible à l'adresse suivante : www.cordis.lu/fr/src/d-021-fr.htm ;

Latitude, longitude en degrés, minutes ou degrés décimaux ;

Bassin versant (1), pour les eaux de surface : nom de la rivière, du lac, du réservoir, de la mer.

2. Données de mesure

Nom du laboratoire.

Catégorie de nucléides.

Type d'appareil.

Valeur de l'activité.

Incertitude.

Type d'incertitude.

Unité de valeur.

Type de valeur.

Date de référence (1), date pour laquelle la valeur d'activité est indiquée.

Débit (1), pour l'eau de rivière.

Rythme de production (1), pour le lait et l'eau potable.

Volume produit ou distribué sur une année, pour l'eau potable.

(1) Uniquement si nécessaire.
