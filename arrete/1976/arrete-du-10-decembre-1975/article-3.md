# Article 3

Pour les groupes électrogènes de puissance supérieure à 8 kW, le niveau de pression sonore global moyen à un mètre déterminé, suivant la méthode prévue dans le code de mesure annexé au présent arrêté, n'excédera pas 85 dBA.

Pour les groupes électrogènes de puissance égale ou inférieure à 8 kW, le niveau de pression sonore maximal mesuré comme prévu dans le code de mesure annexé au présent arrêté n'excédera pas 80 dBA à 7 mètres.
