# Article 4

Sont enregistrées par le traitement les informations ou catégories d'informations suivantes :

Informations relatives au demandeur de l'acte :

- civilité ;

- nom ;

- prénom ;

- qualité ;

- numéro de téléphone ;

- adresse de courrier électronique ;

- code postal ;

- localité ;

- escalier, étage, appartement ;

- immeuble, bâtiment, résidence ;

- numéro et libellé de la voie ;

- lieudit ou boîte postale ;

- motif de la demande (mention facultative).

Informations relatives à l'identité de la personne que l'acte concerne :

- informations exigées par le décret n° 62-921 du 3 août 1962 susvisé pour l'obtention d'une copie intégrale ou d'un extrait d'acte de naissance avec ou sans filiation.

Informations relatives à l'identité des agents habilités à traiter les demandes :

- civilité ;

- nom ;

- prénom ;

- qualité ;

- numéro de téléphone ;

- adresse de courrier électronique ;

- identifiant ;

- mot de passe.
