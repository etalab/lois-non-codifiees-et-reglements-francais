# Article 7

Groupe PC (physique-chimie).

Epreuves écrites d'admissibilité :

1. Composition de mathématiques (durée : quatre heures ; coefficient 4) ;

2. Composition de physique (durée : cinq heures ; coefficient 5) ;

3. Composition de chimie (durée : cinq heures ; coefficient 5).

Epreuves écrites d'admission :

1. Français (durée : quatre heures ; coefficient 2) ;

2. Langue vivante étrangère (durée : deux heures ; coefficient 1,5).

Epreuves orales et pratiques d'admission :

1. Mathématiques (coefficient 4) ;

2. Physique (coefficient 5) ;

3. Chimie (coefficient 5) ;

4. Travaux pratiques de physique (coefficient 4) ;

5. Travaux pratiques de chimie (coefficient 4) ;

6. Langue vivante étrangère (coefficient 1,5) ;

7. Travaux d'initiative personnelle encadrés (coefficient 2).
