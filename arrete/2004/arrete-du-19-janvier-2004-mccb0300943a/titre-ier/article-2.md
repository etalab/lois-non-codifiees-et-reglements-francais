# Article 2

Les étudiants de l'école sont recrutés par concours :

1° Un concours général et un concours international pour le cursus principal ;

2° Un concours spécifique pour le cursus scripte dénommé "concours scripte" ;

3° Un concours spécifique pour le cursus distribution-exploitation dénommé "concours distribution-exploitation" ;

4° Un concours spécifique pour le cursus création de séries télévisuelles, dénommé "concours création de séries télévisuelles".

Le concours général, le concours international, le concours distribution-exploitation et le concours création de séries télévisuelles sont organisés chaque année. Le concours scripte peut être organisé tous les ans.
