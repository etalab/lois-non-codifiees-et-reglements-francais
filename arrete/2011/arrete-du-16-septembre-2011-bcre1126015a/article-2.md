# Article 2

Les informations à caractère personnel traitées sont les suivantes :

I. - Concernant les informations sur le contribuable :

― informations sur l'identification du contribuable (nom, prénom, date de naissance, numéro FIP).

II. - Concernant les informations sur le suivi des mouvements des déclarations papier :

― année de la campagne ;

― identifiant de la déclaration ;

― type de document (déclaration ISF) ;

― code SAGES de la structure de départ/d'arrivée de la déclaration ;

― code direction ;

― nom du site ;

― type de site ;

― date et heure de l'enregistrement types d'anomalies des documents ou du transport.

III. - Concernant les données d'identification et d'habilitation des agents :

― login ;

― mot de passe.
