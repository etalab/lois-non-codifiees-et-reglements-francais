# Article 1

<div id="LEGIARTI000006523807_corps">Pour l'application de la présente loi : <br/>
<br/>1° Le " navire ” est défini à l'article L. 5000-2 du code des transports ; <br/>
<br/>2° L'" armateur ”, l'" entreprise d'armement maritime ”, le " marin ” et les " gens de mer ” sont définis à l'article L. 5511-1 du même code ; <br/>
<br/>3° Le " bord ” est défini à l'article L. 5511-2 du même code ; <br/>
<br/>4° L'" équipage ” est défini à l'article L. 5511-3 du même code ; <br/>
<br/>5° Le " capitaine ”, l'" officier ” et le " maître ” sont définis à l'article L. 5511-4 du même code ; <br/>
<br/>6° Le " passager ” est défini à l'article L. 5511-5 du même code.<br/>
<br/>
</div>
