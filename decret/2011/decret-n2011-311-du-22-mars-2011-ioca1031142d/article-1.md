# Article 1

― Saint-Paul ;

― Noë-les-Mallets ;

― Bathelémont-lès-Bauzemont ;

― Saint-Just ;

― Hédé ;

― Puycelci ;

― Croix-Fonsommes ;

― Fonsommes ;

― Saint-Dionizy,

prennent respectivement le nom de :

― Saint-Paul-de-Vence ;

― Noé-les-Mallets ;

― Bathelémont ;

― Saint-Just-d'Ardèche ;

― Hédé-Bazouges ;

― Puycelsi ;

― Croix-Fonsomme ;

― Fonsomme ;

― Saint-Dionisy.
