# Article 2

La production de lait, la fabrication et l'affinage des fromages sont effectués dans l'aire géographique délimitée par les communes suivantes :

Département du Cher

Arrondissement de Bourges :

Canton des Aix-d'Angillon : toutes les communes ;

Canton d'Argent-sur-Sauldre : les communes d'Argent-sur-Sauldre et Blancafort ;

Canton d'Aubigny-sur-Nère : les communes d'Aubigny-sur-Nère et Oizon ;

Canton de Baugy : toutes les communes ;

Canton de Bourges : la commune de Bourges ;

Canton de La Chapelle-d'Angillon : toutes les communes ;

Canton de Charost : les communes de Civray, Lunery, Mareuil-sur-Amon, Morthomiers, Plou, Poisieux, Primelles, Saint-Florent-sur-Cher, Le Subdray et Villeneuve-sur-Cher ;

Canton d'Henrichemont : toutes les communes ;

Canton de Léré : toutes les communes ;

Canton de Levet : toutes les communes ;

Canton de Lury-sur-Arnon : les communes de Cerbois, Lazenay, Limeux, Preuilly et Quincy ;

Canton de Mehun-sur-Yèvre : toutes les communes ;

Canton de Saint-Doulchard : toutes les communes ;

Canton de Saint-Martin-d'Auxigny : toutes les communes ;

Canton de Sancergues : toutes les communes ;

Canton de Sancerre : toutes les communes ;

Canton de Vailly-sur-Sauldre : toutes les communes ;

Canton de Vierzon II : les communes de Nançay, Neuvy-sur-Barangeon, Saint-Laurent et Vouzeron.

Arrondissement de Saint-Amand-Montrond :

Canton de Châteauneuf-sur-Cher : les communes de Corquoy et Serruelles ;

Canton de Dun-sur-Auron : les communes de Bussy, Lantan, Osmery, Raymond, Saint-Denis-de-Palin et Saint-Germain-des-Bois ;

Canton de Nérondes : toutes les communes.

Département du Loiret

Arrondissement de Montargis :

Canton de Briare : les communes de Bonny-sur-Loire, Faverelles, Ousson-sur-Loire et Thou ;

Canton de Châtillon-sur-Loire : toutes les communes ;

Canton de Gien : les communes de Coullons, Poilly-lès-Gien, Saint-Brisson-sur-Loire et Saint-Martin-sur-Ocre.

Arrondissement d'Orléans :

Canton de Sully-sur-Loire : la commune de Cerdon.

Département de la Nièvre

Arrondissement de Cosne-sur-Loire :

Canton de La Charité-sur-Loire : les communes de La Charité-sur-Loire, Narcy, Raveau et Varennes-lès-Narcy ;

Canton de Cosne-Cours-sur-Loire Nord : toutes les communes ;

Canton de Cosne-Cours-sur-Loire Sud : toutes les communes ;

Canton de Donzy : la commune de Donzy ;

Canton de Pouilly-sur-Loire : les communes de Bulcy, Garchy, Mesves-sur-Loire, Pouilly-sur-Loire, Saint-Andelain, Saint-Laurent-L'Abbaye, Saint-Martin-sur-Nohain, Saint-Quentin-sur-Nohain, Suilly-la-Tour et Tracy-sur-Loire ;

Canton de Saint-Amand-en-Puisaye : les communes d'Arquian et Saint-Vérain.
