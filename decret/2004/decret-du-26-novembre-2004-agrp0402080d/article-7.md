# Article 7

1° Pour les vins rouges et rosés, le rendement de base visé à l'article D. 641-73 du code rural est fixé à 60 hectolitres par hectare. Le rendement butoir visé à l'article D. 641-76 du code rural est fixé à 65 hectolitres par hectare. Le rendement maximum de production visé à l'article D. 641-78 du code rural est fixé à 72 hectolitres par hectare.

2° Pour les vins blancs, le rendement de base visé à l'article D. 641-73 du code rural est fixé à 58 hectolitres à l'hectare. Le rendement butoir prévu à l'article D. 641-76 du code rural est fixé à 70 hectolitres à l'hectare.

3° Le bénéfice de l'appellation d'origine contrôlée "Beaujolais Villages" ou "Beaujolais" suivie du nom de la commune d'origine ne peut être revendiqué sur des parcelles qui n'ont pas été totalement récoltées.
