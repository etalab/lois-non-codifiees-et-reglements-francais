# Article 8

Les épreuves d'admissibilité comportent les épreuves suivantes :

1° Deux épreuves de français :

a) Une composition française sur un sujet d'ordre général (durée :

deux heures ; coefficient 2) ;

b) Une rédaction d'un rapport de synthèse (durée : deux heures ; coefficient 2).

2° Une épreuve de mathématiques :

Deux problèmes dans la même option au choix du candidat lors du dépôt de candidature :

Soit de mathématiques traditionnelles, soit de mathématiques modernes (durée : quatre heures ; coefficient 4).

3° Une épreuve choisie par le candidat parmi les deux épreuves suivantes :

- lever de plan (durée : deux heures ; coefficient 3) et nivellement (durée : une heure ; coefficient 1) ;

- techniques sanitaires (durée : deux heures ; coefficient 4).

4° Une épreuve choisie par le candidat parmi les deux épreuves suivantes :

- dessin (durée : six heures ; coefficient 4), établissement d'un avant-projet sommaire mettant en évidence les qualités techniques du candidat :

- soit sur les travaux publics (V.R.D.) ;

- soit sur les bâtiments ;

- soit sur les espaces verts ;

- chimie (durée : deux heures ; coefficient 4).

5° Une épreuve de droit et pratique du service comprenant deux options relatives, au choix du candidat, soit au domaine de l'urbanisme, soit au domaine sanitaire (durée : deux heures ; coefficient 2).
