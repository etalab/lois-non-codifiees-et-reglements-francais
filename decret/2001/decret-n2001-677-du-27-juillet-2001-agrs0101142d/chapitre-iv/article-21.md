# Article 21

Le plafond de l'exonération prévue par l'article L. 731-13 du code rural est fixé à :

14 949 F pour les jeunes chefs d'exploitation agricole bénéficiant d'une exonération de 65 % ;

12 649 F pour les jeunes chefs d'exploitation agricole bénéficiant d'une exonération de 55 % ;

8 050 F pour les jeunes chefs d'exploitation agricole bénéficiant d'une exonération de 35 % ;

5 750 F pour les jeunes chefs d'exploitation agricole bénéficiant d'une exonération de 25 % ;

3 450 F pour les jeunes chefs d'exploitation agricole bénéficiant d'une exonération de 15 %.
