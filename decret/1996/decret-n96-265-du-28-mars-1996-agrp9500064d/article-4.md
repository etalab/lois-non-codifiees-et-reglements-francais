# Article 4

La taxe à la première vente est due par les vendeurs de semences et plants à l'occasion de la première vente en France de ces produits, qu'il s'agisse de semences et plants certifiés ou non. Son montant est, par quintal, au plus égal à :

16,74 F pour les semences certifiées de céréales hybrides ;

2,66 F pour les semences certifiées de céréales à paille ;

4,35 F pour les semences de base de céréales à paille ;

26,07 F pour les semences de maïs hybrides simples ;

20,40 F pour les semences de maïs hybrides trois voies ;

16,06 F pour les semences de maïs hybrides doubles ;

29,15 F pour les semences de sorgho ;

10,31 F pour les semences certifiées de vesces, féveroles et sainfoin ;

9,05 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

41,80 F pour les semences de graminées fourragères et les autres semences fourragères en espèces pures ;

56,10 F pour les semences de graminées fourragères et les autres semences fourragères, en mélange ;

67,10 F pour les semences de betteraves sucrières ;

13,97 F pour les semences de betteraves fourragères ;

41,25 F pour les semences de chicorée industrielle ;

8,47 F pour les plants de pommes de terre ;

24,89 F pour les semences de lin ;

57,66 F pour les semences de chanvre ;

26,18 F pour les semences de soja et de ricin ;

53,02 F pour les semences de moutarde commerciale ;

145,20 F pour les semences de tournesol hybride ;

62,53 F pour les semences de tournesol population ;

69,58 F pour les colzas hybrides ;

63,26 F pour les autres semences oléagineuses ;

7,45 F pour les semences standard de pois, lentilles et fèves ;

9,46 F pour les semences standard de haricots ;

12,24 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,84 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le montant est au plus égal à 106,70 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines, au plus égal à 78,10 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.

La taxe à la première vente est mise en recouvrement directement par le groupement auprès des vendeurs de semences et plants.
