# Article 1

Le Massif vosgien comprend :

REGION DE FRANCHE-COMTE

Département de la Haute-Saône

Les cantons ayant une partie de leur territoire classée en zone de montagne, ainsi que :

Les communes de Belverne, Chenebier, Courmont, Etobon, dans le canton d'Héricourt-Ouest ;

La commune de Fougerolles dans le canton de Saint-Loup-sur-Semouse.

Département du territoire de Belfort

Les cantons ayant une partie de leur territoire classée en zone de montagne.

Dans le canton d'Offemont, la commune d'Eloie.

REGION DE LORRAINE

Département des Vosges

Dans le canton de Bruyères, les communes de Beaumenil, Bruyères, Champ-le-Duc, Chenimenil, Deycimont, Docelles, Fays, Fimenil, Laval-sur-Vologne, Laveline-Devant-Bruyères, Laveline-du-Haux, Lepanges-sur-Vologne, La Neuveville-devant-Lépanges, Prey, Le Roulier, Xamontarupt ;

Dans le canton de Rambervillers, la commune de Saint-Benoît-la-Chipotte ;

Ainsi que les autres cantons ayant tout ou partie de leur territoire classé en zone de montagne.

Département de Meurthe-et-Moselle

Les cantons de Badonvillers et de Cirey-sur-Vezouze ;

Dans le canton de Baccarat, les communes de Baccarat, Bertrichamps, Deneuvre, Lachapelle, Merviller, Thiaville-sur-Meurthe, Vacqueville, Veney.

Département de la Moselle    Les cantons de Bitche et Volmunster ;

Ainsi que :

Dans le canton de Lorquin, les communes de Abreschviller, Lafrimbolle, Saint-Quirin, Turquestein-Blanrupt, Vasperviller et Voyer ;

Dans le canton de Phalsbourg, les communes de Arzviller, Dabo, Danne-et-Quatre-Vents, Dannelbourg, Garrebourg, Hultehouse, Lutzelbourg, Phalsbourg, Saint-Louis, Vilsbert, Haselbourg, Guntzviller, Henridorff ;

Dans le canton de Rohrbach-lès-Bitche, les communes de Enchenberg, Lambach, Montbronn, Rahling, Siersthal et Soucht ;

Dans le canton de Sarrebourg, les communes de Harreberg, Hommert, Plaine-de-Walsch, Trois-Fontaines et Walscheid ;

Les communes de Métairies, Saint-Quirin, Niderhoff et Hartzviller.

REGION D'ALSACE

Département du Bas-Rhin

Les communes classées en zone de montagne ;

Les cantons de la Petite-Pierre, Saales, Schirmeck, Ville, en entier ;

Ainsi que les communes suivantes :

Le canton de Sélestat : communes d'Orschwiller, Scherwiller et La Vancelle ;

Le canton de Barr : communes d'Andlau, Barr, Dambach-la-Ville, Heiligenstein, du Hohwald et de Reichsfeld ;

Le canton de Kosheim : communes de Boersch, Gkendelbruch, Mollkirch Ottrott, Rosenwiller, Rosheim, Saint-Nabor ;

Le canton de Molsheim : communes de Dinsheim, Gresswiller, Heiligenberg, Lutzelhouse, Muhlbach-sur-Bruche, Niederhaslach, Oberhaslach, Still, Urmatt ;

Le canton de Wasselonne : communes de Cosswiller, Romanswiller, Wangenbourg-Engenthal et Westhoffen ;

Le canton de Marmoutier : communes de Allenwiller, Birkenwald, Dimbsthal, Haegen, Hengwiller, Reinhardsmunster, Salenthal ;

Le canton de Saverne : communes d'Eckartswiller, Ernolsheim-lès-Saverne, Ottersthal, Saint-Jean-lès-Saverne, Saverne ;

Le canton de Bouxwiller : communes de Dossenheim-sur-Zinsel, Ingwiller, Neuwiller-lès-Saverne, Weinbourg et Weiterswiller ;

Le canton de Drulingen : communes de Diemeringen, Volksberg, Waldhambach, Weislingen ;

Le canton de Sarre-Union : communes de Butten et de Ratzwiller ;    Le canton de Niederbronn : communes de Dambach, Niederbronn-les-Bains, Oberdronn-Zinswiller, Offwiler, Rothbach, Reichshoffen, Winstein ;

Le canton de Woerth : les communes de Frofswiller, Goersdorf, Lampertsloch, Langensoultzbach, Mitschdorf, Preuschdorf, Woerth ;

Le canton de Soultz-sous-Forêt : communes de Drachenbronn, Keffenach, Kutzenhausen, Lubsann, Memmelshoffen, Merkwiller-Péchelbronn ;

Le canton de Wissembourg : communes de Cleebourg, Climbach, Lembach, Niedersteinbach, Obersteinbach, Rott, Wingen, Wissembourg.

Département du Haut-Rhin

Les communes classées en zone de montagne ;

Les cantons de Guebwiller, Lapoutroie, Masevaux, Munster, Saint-Amarin, Sainte-Maire-aux-Mines, Thann, Wintzenheim, Ferrette ;    Ainsi que les communes suivantes :

Le canton de Cernay : communes de Steinbach, Uffholtz et Wattwiller ;

Le canton de Soultz : communes de Jungholtz, Hartmannswiller, Soultz et Wuenheim ;

Le canton de Rouffach : communes de Gueberschwihr, Hattstatt, Pfaffenheim, Saultzmatt, Westhalten ;

Le canton de Kaysersberg : communes d'Ammerschuihr, Katzenthal, Kaysersberg, Kientzheim, Niedermoschwihr, Riquewihr ;

Le canton de Ribeauvillé : communes de Bergheim, Hunawihr, Ribeauvillé, Rodern, Saint-Hippolyte et Thannenkirch.
