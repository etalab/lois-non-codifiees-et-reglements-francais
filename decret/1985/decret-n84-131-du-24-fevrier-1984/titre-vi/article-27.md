# Article 27

L'avancement d'échelon s'effectue selon les durées suivantes :

12e échelon : quatre ans ;

11e échelon : deux ans ;

10e échelon : deux ans ;

9e échelon : deux ans ;

8e échelon : deux ans ;

7e échelon : deux ans ;

6e échelon : deux ans ;

5e échelon : deux ans ;

4e échelon : deux ans ;

3e échelon : deux ans ;

2e échelon : un an ;

1er échelon : un an.

L'avancement d'échelon est prononcé par le préfet.
