# Article Annexe

1. Ingrédients ne nécessitant pas de mention dans la dénomination de vente des produits finis.

Noms et conditions d'emploi.

Eau de qualité alimentaire : dans tous les produits définis au titre Ier.

Jus de fruits : dans le produit dénommé "confiture".

Jus de fruits rouges : dans le produit dénommé "confiture extra", lorsqu'il est obtenu à partir de l'un ou plusieurs des fruits suivants : fraises, framboises, groseilles à maquereaux, groseilles rouges et prunes.

Jus de betteraves rouges : dans les produits dénommés "confiture" et "gelée", lorsqu'ils sont obtenus à partir de l'un ou plusieurs des fruits suivants : fraises, framboises, groseilles à maquereaux, groseilles rouges et prunes.

Huiles essentielles d'agrumes : dans le produit dénommé "marmelade".

Huiles et graisses comestibles : en tant qu'agent anti-moussant, dans tous les produits définis au titre Ier.

2. Ingrédients devant être mentionnés dans la dénomination de vente des produits finis.

a) Ingrédients alimentaires en quantité suffisante pour influencer le goût :

- jus et/ou écorces d'agrumes dans les produits obtenus à partir d'autres fruits : dans les produits dénommés "confiture" et "confiture extra" ;

- spiritueux ;

- vin et vin de liqueur ;

- noix, noisettes, amandes ;

- miel, miel de pâtisserie ;

- herbes ;

- épices ;

... Dans tous les produits définis au titre Ier de l'annexe.

- fruits séchés : dans les crèmes de fruits à coque autres que la crème de marrons.

- arôme naturel de rose : 5 g/kg dans le confit de pétales de roses.

b) Ecorces d'agrumes :

- feuilles de Pelargonium odoratissimum : dans les produits dénommés "confiture", "confiture extra", "gelée", "gelée extra", lorsqu'ils sont obtenus à partir de coings.

c) Vanille : dans les produits dénommés "confiture", "confiture extra", "gelée", "gelée extra", lorsqu'ils sont obtenus à partir de pommes, coings ou cynorrhodrons, ainsi que dans les produits dénommés "crème de pruneaux", "crème de marrons", ou d'autres fruits à coque.
