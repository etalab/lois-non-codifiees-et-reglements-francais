# Article 16

Le plafond de l'exonération prévue à l'article L. 731-13 du code rural est fixé à :

1 565,95 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 65 % ;

1 325,04 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 55 % ;

843,21 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 35 % ;

602,29 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 25 % ;

361,37 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 15 %.
