# Article Annexe art. 25

Classification des contrôles de laboratoire selon leur caractère contraignant.

Les essais prévus selon l'article 21 sont répartis par type selon la classification suivante :

Type de l'essai :

Type 1 (essais normaux).

Périodicité des contrôles : Systématique.

Résultat = Décision :

Quel qu'il soit, le résultat emporte la décision.

Type 2 (essais de longue durée).

Périodicité : systématique.

Résultat = Décision :

La décision est prise sans attendre le résultat de l'essai si les résultats précédents sont conformes ; si le résultat de l'essai est non conforme, l'essai classe type 2 passe en type 1.

Type 3 (essais incitatifs).

Périodicité : systématique.

Résultat = Décision :

Le résultat n'intervient pas dans la décision, il constitue un objectif.

Type 4 (essais occasionnels).

Périodicité : variable.

Résultat = Décision :

Le résultat entraîne la décision.
