# Article 16

Le plafond de l'exonération prévue à l'article L. 731-13 du code rural est fixé à :

1 542,68 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 65 % ;

1 305,35 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 55 % ;

830,68 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 35 % ;

593,34 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 25 % ;

356 Euros pour les jeunes agriculteurs bénéficiant d'une exonération de 15 %.
