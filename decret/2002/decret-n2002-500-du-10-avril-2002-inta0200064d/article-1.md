# Article 1

Les communes de :

- Bassilac ;

- Cirière ;

- Grateloup ;

- Grindorff ;

- Mornand ;

- Plouézoch ;

- Saint-Loup-Champagne ;

- Saint-Montant ;

- Saint-Sulpice,

prennent respectivement le nom de :

- Bassillac ;

- Cirières ;

- Grateloup-Saint-Gayrand ;

- Grindorff-Bizing ;

- Mornand-en-Forez ;

- Plouezoc'h ;

- Saint-Loup-en-Champagne ;

- Saint-Montan ;

- Saint-Sulpice-de-Pommeray.
