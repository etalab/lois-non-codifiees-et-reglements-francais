# Article 4

La taxe à la première vente est due par les vendeurs de semences et plants à l'occasion de la première vente en France de ces produits, qu'il s'agisse de semences et plants certifiés ou non. Son montant est, par quintal, au plus égal à :

14,26 F pour les semences certifiées de céréales hybrides ;

2,32 F pour les semences certifiées de céréales à paille ;

3,74 F pour les semences de base de céréales à paille ;

22,50 F pour les semences de maïs hybrides simples ;

17,65 F pour les semences de maïs hybrides trois voies ;

13,86 F pour les semences de maïs hybrides doubles ;

25,10 F pour les semences de sorgho ;

8,92 F pour les semences certifiées de vesces, féveroles et sainfoin ;

8,63 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

36,10 F pour les semences de graminées fourragères ;

36,10 F pour les autres semences fourragères ;

58,08 F pour les semences de betteraves sucrières ;

12,08 F pour les semences de betteraves fourragères ;

35,68 F pour les semences de chicorée industrielle ;

7,39 F pour les plants de pommes de terre ;

21,45 F pour les semences de lin ;

49,85 F pour les semences de chanvre ;

22,82 F pour les semences de soja ;

45,80 F pour les semences de moutarde commerciale ;

128,50 F pour les semences de tournesol hybride ;

54,11 F pour les semences de tournesol population ;

54,45 F pour les autres semences oléagineuses ;

6,40 F pour les semences standard de pois, lentilles et fèves ;

8,10 F pour les semences standard de haricots ;

10,50 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,45 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le montant est au plus égal à 93,68 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines, au plus égal à 68 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.

La taxe à la première vente est mise en recouvrement directement par le groupement auprès des vendeurs de semences et plants.
