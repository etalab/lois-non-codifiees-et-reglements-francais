# Article 1

Les communes de :

― Boën ;

― Châlons ;

― Saint-Martin ;

― Peumérit ;

― Braisnes ;

― Saints ;

― Saint-Denis,

prennent respectivement le nom de :

― Boën-sur-Lignon ;

― Chalon ;

― Saint-Martin-de-Pallières ;

― Peumerit ;

― Braisnes-sur-Aronde ;

― Saints-en-Puisaye ;

― Saint-Denis-lès-Sens.
