# Article 1

Les communes de :

― Saint-Ouen (Charente-Maritime) ;

― Le Bouchet (Haute-Savoie) ;

― Saint-Sulpice (Tarn) ;

― Saint-Alban-des-Hurtières (Savoie) ;

― Saint-Georges-des-Hurtières (Savoie) ;

― Saint-Amant (Charente) ;

― Onesse-et-Laharie (Landes) ;

― Meyssies (Isère),

prennent respectivement le nom de :

― Saint-Ouen-la-Thène ;

― Le Bouchet-Mont-Charvin ;

― Saint-Sulpice-la-Pointe ;

― Saint-Alban-d'Hurtières ;

― Saint-Georges-d'Hurtières ;

― Saint-Amant-de-Montmoreau ;

― Onesse-Laharie ;

― Meyssiez.
