# Article 2

Les agents spécialistes exercent notamment les fonctions suivantes : aide-concierge, aide de cuisine, aide-magasinier, agent d'entretien, agent de réfectoire, concierge-vaguemestre-standardiste, agent de sécurité, veilleur.

Seuls peuvent exercer les fonctions de concierge-vaguemestre-standardiste, lorsqu'il s'agit d'un poste double, les agents spécialistes dont le conjoint satisfait aux conditions requises pour exercer celles d'aide-concierge.
