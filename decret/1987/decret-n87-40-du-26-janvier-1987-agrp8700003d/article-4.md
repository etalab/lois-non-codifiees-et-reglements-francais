# Article 4

La taxe à la première vente est due par les vendeurs de semences et plants [*fait générateur*] à l'occasion de la première vente en France de ces produits, qu'il s'agisse de semences et plants certifiés en France ou non.

Son montant est, par quintal, au plus égal à :

13,45 F pour les semences certifiées de céréales hybrides ;

1,82 F pour les semences certifiées de céréales à paille ;

3,22 F pour les semences de base de céréales à paille ;

21,16 F pour les semences de maïs hybrides simples ;

16,46 F pour les semences de maïs hybrides trois voies ;

12,87 F pour les semences de maïs hybrides doubles ;

21,67 F pour les semences de sorgho ;

6,26 F pour les semences certifiées de vesces, féveroles et sainfoin ;

7,90 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

25,32 F pour les semences de graminées fourragères ;

25,32 F pour les autres semences fourragères ;

54,23 F pour les semences de betteraves sucrières ;

9,56 F pour les semences de betteraves fourragères ;

32,26 F pour les semences de chicorée industrielle ;

7,25 F pour les plants de pommes de terre ;

17,40 F pour les semences de lin ;

47,65 F pour les semences de chanvre ;

19,60 F pour les semences de soja ;

38,55 F pour les semences de moutarde commerciale ;

78,20 F pour les semences de tournesol hybride ;

52,28 F pour les semences de tournesol population ;

46,48 F pour les autres semences oléagineuses ;

0,56 F pour les semences standard de pois, lentilles et fèves ;

0,56 F pour les semences standard de haricots ;

0,56 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles ;

2,35 F pour les plants de fraisiers certifiés présentés en unités de 1 000 pieds.

Le montant est au plus égal à 82,55 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères présentées en unités de 100 000 graines, et au plus égal à 59,15 F par cent unités pour les semences de chicorée industrielle présentées en unités de 100 000 graines.

La taxe à la première vente est mise en recouvrement directement par le groupement auprès des vendeurs de semences et plants.
