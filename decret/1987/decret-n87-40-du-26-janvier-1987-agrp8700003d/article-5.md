# Article 5

La taxe additionnelle est assise [*assiette*] sur les mêmes bases que la taxe à la première vente et recouvrée [*recouvrement*] de la même façon.

Elle n'est due toutefois que pour les semences et plants figurant ci-dessous ; son montant est, par quintal, au plus égal à :

0,38 F pour les semences de céréales ;

0,48 F pour les semences de maïs ;

2,43 F pour les semences de sorgho ;

2,32 F pour les semences certifiées de vesces, féveroles et sainfoin ;

1,95 F pour les semences certifiées de lupin, pois protéagineux et autres pois fourragers ;

9,33 F pour les autres semences fourragères ;

2,00 F pour les semences de betteraves et de chicorée industrielle ;

0,11 F pour les plants de pommes de terre ;

3,17 F pour les semences de lin ;

2,32 F pour les semences de soja ;

44,75 F pour les semences de tournesol ;

5,80 F pour les semences de moutarde commerciale ;

5,80 F pour les autres semences oléagineuses ;

5,60 F pour les semences standards de pois potagers, fèves et lentilles ;

7,25 F pour les semences standards de haricots ;

9,55 F pour les semences certifiées de pois potagers, haricots, fèves et lentilles.

Le montant est au plus égal à 6,26 F par cent unités pour les semences de betteraves monogermes sucrières et fourragères et de chicorée industrielle présentées en unités de 100 000 graines.
