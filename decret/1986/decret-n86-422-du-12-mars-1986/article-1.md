# Article 1

La liste des cantons mentionnés à l'article 5 de la loi n° 86-29 du 9 janvier 1986 est arrêtée comme suit :

Pour le département de l'Allier

Les cantons de : Commentry, Cusset (deux cantons), Escurolles, Hérisson, Huriel, Marcillat-en-Combraille, Domérat - Montluçon-Nord-Ouest, Montluçon (quatre cantons), Montmarault, Vichy (deux cantons).

Pour le département des Ardennes

Les cantons de : Carignan, Charleville-Centre, Charleville-la-Houillère, Flize, Fumay, Givet, Mézières-Centre-Ouest, Mézières-Est, Monthermé, Mouzon, Nouzonville, Omont, Renwez, Revin, Rocroi, Rumigny, Sedan (trois cantons), Signy-l'Abbaye, Signy-le-Petit, Villers-Semeuse.

Pour le département de l'Ariège

Les cantons de Tarascon, Vicdessos.

Pour le département de l'Aveyron

Les cantons de : Aubin, Capdenac-Gare, Decazeville, Montbazens, Villefranche-de-Rouergue.

Pour le département des Bouches-du-Rhône

Les cantons de : La Ciotat, Istres, Martigues, Port-Saint-Louis-du-Rhône.

Pour le département du Calvados

Les cantons de : Bretteville, Bourguébus, Cabourg, Caen (dix cantons), Cambremer, Dozulé, Mézidon, Ouistreham, Troarn.

Pour le département du Cher

Les cantons de Bourges (cinq cantons), Saint-Doulchard.

Pour le département de la Corrèze

Les cantons de Tulle (quatre cantons).

Pour le département de l'Indre

Les cantons de Saint-Christophe-en-Bazelle, Valençay, Vatan.

Pour le département de la Loire

Les cantons de : Le Chambon-Feugerolles, Firminy, La Grand-Croix, Rive-de-Gier, Saint-Chamond-Nord, Saint-Chamond-Sud, Saint-Etienne (neuf cantons), Saint-Héand, Charlieu, Néronde, Perreux, Roanne-Nord, Roanne-Sud, Saint-Germain-Laval, Saint-Symphorien-de-Lay.

Pour le département de Loir-et-Cher

Les cantons de Lamotte-Beuvron, Mennetou-sur-Cher, Neung-sur-Beuvron, Romorantin (deux cantons), Salbris, Selles-sur-Cher.

Pour le département de Maine-et-Loire

Les cantons d'Angers (huit cantons), Ponts-de-Cé.

Pour les cantons de Meurthe-et-Moselle

Les cantons de : Audun-le-Roman, Briey, Conflans-en-Jarnisy, Homécourt, Herserange, Longuyon, Longwy, Mont-Saint-Martin, Villerupt, Jarville-la-Malgrange, Laxou, Nancy-Est, Neuves-Maisons, Pompey, Saint-Max, Saint-Nicolas-de-Port, Tomblaine, Vandoeuvre-lès-Nancy.

Pour le département de la Meuse

Le canton de : Spincourt.

Pour le département de la Moselle

Les cantons de :  Maiziéres-lés-Metz, Rombas, Marange-Silvange, Cattenom, Metzervisse, Thionville-Est, Thionville-Ouest, Yutz, Algrange, Fameck, Florange, Fontoy, Hayange, Moyeuvre-Grande, Vigy.

Pour le département du Nord

Les cantons de :  Douai (trois cantons), Marchiennes, Anzin, Bouchain, Condé-sur-l'Escaut, Denain, Saint-Amand-les-Eaux-Rive droite, Saint-Amand-les-Eaux-Rive gauche, Valenciennes (trois cantons), Bergues, Bourbourg, Coudekerque-Branche, Dunkerque-Est, Dunkerque-Ouest, Grande-Synthe, Gravelines, Hondchoote, Wormhout, Bavay, Berlaimont, Hautmont, Maubeuge-Nord, Maubeuge-Sud, Trelon.

Pour le département de l'Orne

Les cantons d' Argentan (deux cantons).

Pour le département du Pas-de-Calais

Les cantons de : Béthune-Nord, Béthune-Sud, Bruay-en-Artois, Barlin, Cambrin, Houdain, Noeux-les-Mines, Bully-les-Mines, Carvin, Courrières, Harnes, Hénin-Beaumont, Le Forest, Lens (trois cantons), Wingles, Liévin-Nord, Liévin-Sud, Avion, Rouvroy, Vimy, Calais (quatre cantons), Audruicq.

Pour le département des Hautes-Pyrénées

Les cantons de Tarbes (cinq cantons), Aureilhan, Séméac, Laloubère.

Pour le département de Saône-et-Loire

Les cantons de : Le Creusot-Est, Le Creusot-Ouest, Montcenis, Chagny, Chalon-sur-Saône (quatre cantons), Givry, La Guiche, Montceau-les-Mines-Nord, Montceau-les-Mines-Sud, Montchanin, Saint-Martin-en-Bresse, Toulon-sur-Arroux.

Pour le département du Tarn

Les cantons de : Albi (quatre cantons), Carmaux-Nord, Carmaux-Sud, Gaillac, Monestiés, Valderiés, Villefranche-d'Albigeois.

Pour le département du Var

Les cantons de : Le Beausset, Ollioules, Saint-Mandrier-sur-Mer, La Seyne-sur-Mer, Six-Fours-les-plages, Toulon (neuf cantons), La Valette-du-Var, Sollies-Pont.
